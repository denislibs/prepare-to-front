# Расскажите о свойстве `text-decoration-skip-ink`?

Свойство `text-decoration-skip-ink` в CSS контролирует, как подчеркивание и зачеркивание взаимодействуют с глифами (символами) текста. Это улучшает читаемость, особенно для текста с подстрочными и надстрочными элементами.

## Что такое `text-decoration-skip-ink`?

`text-decoration-skip-ink` определяет, должны ли линии декорации (подчеркивание, зачеркивание) пропускать части глифов, которые опускаются ниже базовой линии или поднимаются выше.

## Значения

### `auto` — Автоматически

```css
.element {
    text-decoration-skip-ink: auto; /* По умолчанию */
}
```

Браузер решает, пропускать ли части глифов.

### `none` — Не пропускать

```css
.element {
    text-decoration-skip-ink: none;
}
```

Линии декорации проходят через все глифы.

### `all` — Пропускать все

```css
.element {
    text-decoration-skip-ink: all;
}
```

Линии декорации пропускают все части глифов, выходящие за базовую линию.

## Примеры использования

### Улучшение читаемости:

```css
.underlined {
    text-decoration: underline;
    text-decoration-skip-ink: auto; /* Пропускать части букв */
}
```

### Для всех декораций:

```css
.text {
    text-decoration: underline;
    text-decoration-skip-ink: all; /* Пропускать все части */
}
```

## Влияние на читаемость

### С `auto` или `all`:

- ✅ Лучшая читаемость
- ✅ Подчеркивание не перекрывает буквы
- ✅ Особенно полезно для букв с хвостами (g, j, p, q, y)

### С `none`:

- ❌ Подчеркивание может перекрывать буквы
- ❌ Может ухудшить читаемость

## Заключение

**`text-decoration-skip-ink` контролирует:**

1. ✅ Взаимодействие декораций с глифами
2. ✅ Читаемость подчеркнутого текста
3. ✅ Визуальное качество декораций

**Значения:**

- `auto` — автоматически
- `none` — не пропускать
- `all` — пропускать все

**Рекомендации:**

- Используйте `auto` или `all` для лучшей читаемости
- Особенно полезно для букв с хвостами

Правильное использование `text-decoration-skip-ink` улучшает читаемость и визуальное качество текста.
