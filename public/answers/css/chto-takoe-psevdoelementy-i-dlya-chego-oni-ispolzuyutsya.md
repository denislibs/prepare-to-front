# Что такое псевдоэлементы? И для чего они используются?

Псевдоэлементы в CSS — это ключевые слова, которые позволяют стилизовать определенные части элементов, которые не существуют в HTML, но могут быть созданы через CSS.

## Что такое псевдоэлементы?

Псевдоэлементы — это способ выбрать и стилизовать части элементов, которые не представлены в HTML-разметке, но могут быть созданы через CSS.

## Синтаксис

Псевдоэлементы используют двойное двоеточие `::` (в CSS3):

```css
селектор::псевдоэлемент {
    свойство: значение;
}
```

**Примечание:** В CSS2 использовалось одно двоеточие `:`, но в CSS3 рекомендуется `::`.

## Основные псевдоэлементы

### 1. **`::before`** — Перед элементом

Создает элемент перед содержимым:

```css
.element::before {
    content: "→ ";
    color: blue;
}
```

### 2. **`::after`** — После элемента

Создает элемент после содержимого:

```css
.element::after {
    content: " ←";
    color: red;
}
```

**Важно:** `::before` и `::after` требуют свойства `content`, даже если оно пустое:

```css
.element::before {
    content: ""; /* Обязательно! */
}
```

### 3. **`::first-line`** — Первая строка

Стилизует первую строку текста:

```css
p::first-line {
    font-weight: bold;
    color: blue;
}
```

### 4. **`::first-letter`** — Первая буква

Стилизует первую букву текста:

```css
p::first-letter {
    font-size: 2em;
    font-weight: bold;
    color: red;
}
```

### 5. **`::selection`** — Выделенный текст

Стилизует выделенный пользователем текст:

```css
::selection {
    background-color: yellow;
    color: black;
}
```

### 6. **`::placeholder`** — Плейсхолдер

Стилизует плейсхолдер в input:

```css
input::placeholder {
    color: #999;
    font-style: italic;
}
```

## Примеры использования

### Декоративные элементы:

```css
.quote::before {
    content: """;
    font-size: 3em;
    color: #ccc;
}

.quote::after {
    content: """;
    font-size: 3em;
    color: #ccc;
}
```

### Иконки:

```css
.button::before {
    content: "✓ ";
    margin-right: 5px;
}
```

### Декоративные линии:

```css
.title::after {
    content: "";
    display: block;
    width: 50px;
    height: 2px;
    background: blue;
    margin-top: 10px;
}
```

### Буквица:

```css
p::first-letter {
    font-size: 3em;
    float: left;
    line-height: 1;
    margin-right: 5px;
}
```

## Комбинация псевдоэлементов

```css
.element::before {
    content: "Начало";
}

.element::after {
    content: "Конец";
}
```

## Ограничения

### Не работают на:

- Замещаемых элементах (`<img>`, `<input type="image">`)
- Элементах `<br>`
- Пустых элементах (без `content`)

## Заключение

**Псевдоэлементы используются для:**

1. ✅ Декоративных элементов
2. ✅ Иконок и символов
3. ✅ Стилизации частей текста
4. ✅ Создания визуальных эффектов

**Основные псевдоэлементы:**
- `::before` — перед элементом
- `::after` — после элемента
- `::first-line` — первая строка
- `::first-letter` — первая буква
- `::selection` — выделенный текст
- `::placeholder` — плейсхолдер

**Важно:**
- `::before` и `::after` требуют `content`
- Используйте `::` (двойное двоеточие) в CSS3

Псевдоэлементы — мощный инструмент для создания декоративных элементов без изменения HTML.
