# Как работают каскадность и наследование в CSS?

Каскадность (cascade) и наследование (inheritance) — это фундаментальные принципы CSS, которые определяют, какие стили применяются к элементам и как они передаются от родителей к потомкам.

## Каскадность (Cascade)

Каскадность определяет порядок применения стилей при конфликтах.

### Порядок приоритета (от низкого к высокому):

1. Браузерные стили (User Agent)
2. Внешние стили (`<link>`)
3. Встроенные стили (`<style>`)
4. Инлайновые стили (`style="..."`)
5. `!important` (переопределяет все)

### Специфичность:

При равном источнике применяется правило с большей специфичностью:

```css
p { color: blue; }        /* Специфичность: 1 */
.text { color: red; }     /* Специфичность: 10 */
#header { color: green; }  /* Специфичность: 100 */
```

## Наследование (Inheritance)

Наследование — это передача стилей от родительских элементов к дочерним.

### Наследуемые свойства:

```css
body {
    font-family: Arial;
    color: #333;
    line-height: 1.5;
}

/* Все элементы внутри body наследуют эти стили */
```

### Ненаследуемые свойства:

```css
body {
    margin: 0;
    padding: 0;
    border: 1px solid black;
}

/* margin, padding, border НЕ наследуются */
```

## Примеры

### Каскадность:

```css
/* Внешний файл */
p { color: blue; }

/* Встроенный стиль */
p { color: red; } /* Победит */

/* Инлайновый */
/* <p style="color: green;">Текст</p> */ /* Победит */
```

### Наследование:

```html
<div class="parent">
    <p>Текст наследует стили parent</p>
</div>
```

```css
.parent {
    color: blue;
    font-size: 16px;
}

/* p наследует color и font-size */
```

## Управление наследованием

### `inherit` — Наследовать:

```css
.child {
    color: inherit; /* Наследует от родителя */
}
```

### `initial` — Начальное значение:

```css
.child {
    color: initial; /* Значение по умолчанию */
}
```

### `unset` — Сброс:

```css
.child {
    color: unset; /* Наследует, если можно, иначе initial */
}
```

## Заключение

**Каскадность:**
- Определяет порядок применения стилей
- Зависит от источника и специфичности
- `!important` переопределяет все

**Наследование:**
- Передача стилей от родителей к потомкам
- Работает для текстовых свойств
- Не работает для layout свойств

**Управление:**
- `inherit` — принудительное наследование
- `initial` — начальное значение
- `unset` — сброс

Понимание каскадности и наследования критично для работы с CSS.
