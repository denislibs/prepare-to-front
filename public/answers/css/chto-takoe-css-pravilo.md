# Что такое CSS-правило?

CSS-правило (CSS rule) — это основная единица CSS, которая определяет, какие стили должны применяться к определенным HTML-элементам. CSS-правило состоит из селектора и блока объявлений.

## Структура CSS-правила

```css
селектор {
    свойство: значение;
    другое-свойство: значение;
}
```

### Компоненты:

1. **Селектор** — определяет, к каким элементам применяются стили
2. **Блок объявлений** — содержится в фигурных скобках `{}`
3. **Объявления** — пары "свойство: значение", разделенные точкой с запятой

## Примеры CSS-правил

### Простое правило:

```css
p {
    color: blue;
    font-size: 16px;
}
```

**Расшифровка:**
- `p` — селектор (все параграфы)
- `{ }` — блок объявлений
- `color: blue;` — объявление (свойство: значение)

### Правило с несколькими объявлениями:

```css
.button {
    background-color: #3498db;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}
```

### Правило с псевдоклассом:

```css
a:hover {
    color: red;
    text-decoration: underline;
}
```

### Правило с медиа-запросом:

```css
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}
```

## Типы селекторов в правилах

### 1. **Селектор по тегу:**

```css
h1 {
    font-size: 32px;
}
```

### 2. **Селектор по классу:**

```css
.text-primary {
    color: blue;
}
```

### 3. **Селектор по ID:**

```css
#header {
    height: 100px;
}
```

### 4. **Комбинированный селектор:**

```css
div.container p {
    margin: 10px;
}
```

### 5. **Селектор по атрибуту:**

```css
[type="text"] {
    border: 1px solid #ccc;
}
```

## Синтаксис объявлений

### Формат объявления:

```css
свойство: значение;
```

### Примеры:

```css
color: blue;                    /* Цвет текста */
font-size: 16px;                /* Размер шрифта */
margin: 10px;                   /* Внешний отступ */
padding: 5px 10px;              /* Внутренний отступ */
background-color: #f0f0f0;      /* Цвет фона */
border: 1px solid black;       /* Граница */
```

### Множественные значения:

```css
margin: 10px 20px 30px 40px;   /* top right bottom left */
padding: 5px 10px;              /* top/bottom left/right */
border: 1px solid #000;        /* width style color */
```

## Группировка правил

### Несколько селекторов с одинаковыми стилями:

```css
h1, h2, h3 {
    font-weight: bold;
    color: #333;
}
```

### Вложенные правила (SCSS/SASS):

```scss
.container {
    width: 100%;
    
    .item {
        padding: 10px;
        
        &:hover {
            background: blue;
        }
    }
}
```

## Комментарии в CSS

```css
/* Это однострочный комментарий */

/*
   Это многострочный
   комментарий
*/

.rule {
    /* Комментарий внутри правила */
    color: blue;
}
```

## Каскадность правил

Правила применяются в определенном порядке:

```css
/* 1. Браузерные стили */
p { margin: 1em 0; }

/* 2. Внешние стили */
p { color: blue; }

/* 3. Встроенные стили */
p { font-size: 16px; }

/* 4. Инлайновые стили */
/* style="color: red;" */

/* 5. !important */
p { color: green !important; }
```

## Специфичность правил

Вес селектора определяет приоритет:

```css
/* Специфичность: 1 */
p { color: blue; }

/* Специфичность: 10 */
.text { color: red; }

/* Специфичность: 100 */
#header { color: green; }

/* Специфичность: 110 */
#header.text { color: purple; }
```

## Примеры практических правил

### Базовое правило:

```css
body {
    font-family: 'Arial', sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f5f5f5;
}
```

### Правило с псевдоклассами:

```css
.button {
    background-color: #3498db;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
}

.button:hover {
    background-color: #2980b9;
}

.button:active {
    background-color: #21618c;
}
```

### Правило с медиа-запросами:

```css
.container {
    width: 100%;
    padding: 20px;
}

@media (min-width: 768px) {
    .container {
        max-width: 750px;
        margin: 0 auto;
    }
}
```

### Правило с анимацией:

```css
@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

.element {
    animation: fadeIn 0.5s ease-in-out;
}
```

## Заключение

**CSS-правило состоит из:**

1. **Селектора** — определяет элементы
2. **Блока объявлений** — фигурные скобки `{}`
3. **Объявлений** — пары "свойство: значение"

**Структура:**
```css
селектор {
    свойство: значение;
    другое-свойство: значение;
}
```

**Ключевые моменты:**
- ✅ Каждое правило определяет стили для определенных элементов
- ✅ Объявления разделяются точкой с запятой
- ✅ Правила применяются в порядке каскадности
- ✅ Специфичность определяет приоритет

CSS-правила — это основа стилизации веб-страниц, позволяющая контролировать внешний вид элементов.
