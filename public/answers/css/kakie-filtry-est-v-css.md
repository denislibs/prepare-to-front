# Какие фильтры есть в CSS?

CSS фильтры позволяют применять графические эффекты к элементам, такие как размытие, яркость, контрастность и другие. Фильтры применяются к элементу до его отображения.

## Функции фильтров

### 1. **`blur()` — Размытие**

```css
.element {
    filter: blur(5px);
}
```

### 2. **`brightness()` — Яркость**

```css
.element {
    filter: brightness(1.5); /* Увеличить яркость на 50% */
    filter: brightness(0.5); /* Уменьшить яркость на 50% */
}
```

### 3. **`contrast()` — Контрастность**

```css
.element {
    filter: contrast(1.5); /* Увеличить контраст */
    filter: contrast(0.5); /* Уменьшить контраст */
}
```

### 4. **`grayscale()` — Оттенки серого**

```css
.element {
    filter: grayscale(100%); /* Полностью серый */
    filter: grayscale(50%); /* Частично серый */
}
```

### 5. **`hue-rotate()` — Поворот оттенка**

```css
.element {
    filter: hue-rotate(90deg); /* Повернуть оттенок на 90 градусов */
}
```

### 6. **`invert()` — Инверсия**

```css
.element {
    filter: invert(100%); /* Полная инверсия */
    filter: invert(50%); /* Частичная инверсия */
}
```

### 7. **`opacity()` — Прозрачность**

```css
.element {
    filter: opacity(0.5); /* 50% прозрачности */
}
```

### 8. **`saturate()` — Насыщенность**

```css
.element {
    filter: saturate(2); /* Увеличить насыщенность в 2 раза */
    filter: saturate(0.5); /* Уменьшить насыщенность */
}
```

### 9. **`sepia()` — Сепия**

```css
.element {
    filter: sepia(100%); /* Полный эффект сепии */
    filter: sepia(50%); /* Частичный эффект */
}
```

### 10. **`drop-shadow()` — Тень**

```css
.element {
    filter: drop-shadow(5px 5px 10px rgba(0, 0, 0, 0.5));
}
```

## Комбинирование фильтров

### Несколько фильтров:

```css
.element {
    filter: blur(2px) brightness(1.2) contrast(1.1);
}
```

## Примеры использования

### Hover эффект:

```css
.image {
    filter: grayscale(100%);
    transition: filter 0.3s;
}

.image:hover {
    filter: grayscale(0%);
}
```

### Затемнение:

```css
.overlay {
    filter: brightness(0.5);
}
```

### Размытие фона:

```css
.background {
    filter: blur(10px);
}
```
