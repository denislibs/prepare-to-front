# Что такое плавающие элементы (floats)? Как они работают?

Плавающие элементы (floats) — это CSS свойство, которое позволяет элементу "плавать" влево или вправо, а текст и другие элементы обтекают его. Изначально использовалось для обтекания текста вокруг изображений, позже использовалось для создания макетов.

## Что такое float?

`float` — это CSS свойство, которое перемещает элемент влево или вправо, позволяя другим элементам обтекать его.

## Значения float

```css
.element {
    float: left;   /* Плавание влево */
    float: right;  /* Плавание вправо */
    float: none;   /* Не плавает (по умолчанию) */
}
```

## Как работает float

### Базовое поведение:

```css
img {
    float: left;
    margin-right: 10px;
}
```

**Результат:** Изображение плавает слева, текст обтекает справа.

### Обтекание:

```html
<div class="container">
    <div class="float-left">Плавающий элемент</div>
    <p>Текст обтекает плавающий элемент...</p>
</div>
```

```css
.float-left {
    float: left;
    width: 200px;
}
```

## Очистка float (clear)

### Проблема:

Плавающие элементы выходят из потока, что может ломать макет.

### Решение — `clear`:

```css
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
```

### Или:

```css
.element {
    clear: left;   /* Очистить слева */
    clear: right;  /* Очистить справа */
    clear: both;   /* Очистить с обеих сторон */
}
```

## Использование для макетов (устарело)

### Старый способ:

```css
.sidebar {
    float: left;
    width: 25%;
}

.content {
    float: right;
    width: 75%;
}

.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
```

**Не рекомендуется:** Используйте Flexbox или Grid вместо float для макетов.

## Современное использование

### Только для обтекания текста:

```css
img {
    float: left;
    margin: 0 10px 10px 0;
}
```

## Проблемы float

### 1. **Выход из потока**

Плавающие элементы выходят из нормального потока.

### 2. **Схлопывание родителя**

Родитель может схлопнуться, если все дети плавающие.

### 3. **Сложность очистки**

Требуется clearfix для правильной работы.

## Альтернативы

### Flexbox:

```css
.container {
    display: flex;
    gap: 20px;
}
```

### Grid:

```css
.container {
    display: grid;
    grid-template-columns: 1fr 3fr;
}
```

## Заключение

**Float используется для:**

1. ✅ Обтекания текста вокруг изображений
2. ⚠️ Создания макетов (устарело, используйте Flexbox/Grid)

**Как работает:**
- Элемент плавает влево/вправо
- Текст и другие элементы обтекают его
- Элемент выходит из нормального потока

**Проблемы:**
- Выход из потока
- Схлопывание родителя
- Требуется clearfix

**Рекомендации:**
- Используйте только для обтекания текста
- Используйте Flexbox/Grid для макетов
- Избегайте float для создания макетов

Float — устаревшая техника для макетов, но все еще полезна для обтекания текста вокруг изображений.
