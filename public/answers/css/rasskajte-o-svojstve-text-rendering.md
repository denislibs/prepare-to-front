# Расскажите о свойстве `text-rendering`?

Свойство `text-rendering` в CSS контролирует качество рендеринга текста, оптимизируя баланс между качеством отображения и производительностью. Это свойство особенно полезно для SVG и текста с высокой детализацией.

## Что такое `text-rendering`?

`text-rendering` — это свойство, которое указывает браузеру, как оптимизировать рендеринг текста, балансируя между качеством, скоростью и геометрической точностью.

## Значения

### `auto` — Автоматически

```css
.element {
    text-rendering: auto; /* По умолчанию */
}
```

Браузер выбирает оптимальный режим.

### `optimizeSpeed` — Оптимизация скорости

```css
.element {
    text-rendering: optimizeSpeed;
}
```

Приоритет скорости над качеством. Отключает кернинг и лигатуры.

### `optimizeLegibility` — Оптимизация читаемости

```css
.element {
    text-rendering: optimizeLegibility;
}
```

Приоритет качества и читаемости. Включает кернинг и лигатуры.

### `geometricPrecision` — Геометрическая точность

```css
.element {
    text-rendering: geometricPrecision;
}
```

Геометрическая точность без оптимизаций. Полезно для SVG.

## Примеры использования

### Заголовки:

```css
h1, h2, h3 {
    text-rendering: optimizeLegibility; /* Лучшее качество для заголовков */
}
```

### Мелкий текст:

```css
.small-text {
    text-rendering: optimizeSpeed; /* Скорость для мелкого текста */
    font-size: 12px;
}
```

### SVG текст:

```css
svg text {
    text-rendering: geometricPrecision; /* Точность для SVG */
}
```

## Влияние на производительность

### `optimizeSpeed`:

- ✅ Быстрее рендеринг
- ❌ Меньше качество
- ❌ Нет кернинга

### `optimizeLegibility`:

- ✅ Лучшее качество
- ✅ Кернинг и лигатуры
- ❌ Медленнее рендеринг

### `geometricPrecision`:

- ✅ Точность
- ✅ Полезно для SVG
- ❌ Может быть медленнее

## Заключение

**`text-rendering` контролирует:**

1. ✅ Качество рендеринга текста
2. ✅ Баланс между качеством и скоростью
3. ✅ Использование кернинга и лигатур

**Значения:**

- `auto` — автоматически
- `optimizeSpeed` — скорость
- `optimizeLegibility` — читаемость
- `geometricPrecision` — точность

**Рекомендации:**

- Используйте `optimizeLegibility` для заголовков
- Используйте `optimizeSpeed` для мелкого текста
- Используйте `geometricPrecision` для SVG

Правильное использование `text-rendering` улучшает качество и производительность текста.
