# Расскажите о свойстве `scrollbar-gutter`?

Свойство `scrollbar-gutter` в CSS резервирует пространство для полосы прокрутки, предотвращая сдвиг контента при появлении или исчезновении скроллбара. Это улучшает стабильность макета.

## Что такое `scrollbar-gutter`?

`scrollbar-gutter` резервирует пространство для полосы прокрутки, даже когда скроллбар не отображается, предотвращая сдвиг контента.

## Значения

### `auto` — Автоматически

```css
.element {
    scrollbar-gutter: auto; /* По умолчанию */
}
```

Пространство резервируется только когда скроллбар видим.

### `stable` — Стабильно

```css
.element {
    scrollbar-gutter: stable;
}
```

Пространство всегда резервируется, даже когда скроллбар скрыт.

### `stable both-edges` — С обеих сторон

```css
.element {
    scrollbar-gutter: stable both-edges;
}
```

Пространство резервируется с обеих сторон (для RTL и LTR).

## Примеры использования

### Предотвращение сдвига:

```css
body {
    scrollbar-gutter: stable; /* Резервирует место для скроллбара */
}
```

### Для контейнеров:

```css
.scrollable-container {
    overflow-y: auto;
    scrollbar-gutter: stable;
}
```

## Проблема, которую решает

### Без `scrollbar-gutter`:

Контент сдвигается при появлении/исчезновении скроллбара.

### С `scrollbar-gutter`:

Пространство всегда зарезервировано, контент не сдвигается.

## Заключение

**`scrollbar-gutter` используется для:**

1. ✅ Предотвращения сдвига контента
2. ✅ Стабильности макета
3. ✅ Резервирования места для скроллбара

**Значения:**

- `auto` — автоматически
- `stable` — стабильно
- `stable both-edges` — с обеих сторон

**Рекомендации:**

- Используйте `stable` для предотвращения сдвига
- Особенно полезно для модальных окон
- Улучшает стабильность макета

Правильное использование `scrollbar-gutter` предотвращает сдвиг контента и улучшает UX.
