# Что такое CSS-атрибут (`attr`)?

CSS-функция `attr()` позволяет использовать значения HTML-атрибутов в CSS. Это создает связь между HTML и CSS, позволяя динамически использовать данные из атрибутов.

## Что такое `attr()`?

`attr()` — это CSS-функция, которая извлекает значение атрибута HTML-элемента и использует его в CSS.

## Использование

### Базовый синтаксис:

```css
.element::before {
    content: attr(data-text);
}
```

### С типом и fallback:

```css
.element {
    width: attr(data-width px, 100px); /* Тип и fallback */
}
```

## Примеры использования

### С псевдоэлементами:

```html
<div data-label="Важно">Контент</div>
```

```css
div::before {
    content: attr(data-label) ": ";
    font-weight: bold;
}
```

### С `content`:

```html
<a href="/page" data-title="Перейти на страницу">Ссылка</a>
```

```css
a::after {
    content: " (" attr(data-title) ")";
}
```

### С CSS переменными:

```html
<div data-color="blue">Элемент</div>
```

```css
div {
    --color: attr(data-color);
    color: var(--color);
}
```

## Ограничения

### Поддержка типов:

Не все типы поддерживаются во всех браузерах.

### Использование:

В основном используется с `content` в псевдоэлементах.

## Заключение

**`attr()` используется для:**

1. ✅ Извлечения значений атрибутов
2. ✅ Использования в CSS
3. ✅ Связи HTML и CSS

**Использование:**

- С `content` в псевдоэлементах
- С CSS переменными
- Для динамических значений

**Ограничения:**

- Ограниченная поддержка типов
- В основном с `content`

**Рекомендации:**

- Используйте с псевдоэлементами
- Проверяйте поддержку браузеров
- Используйте fallback значения

Правильное использование `attr()` создает более гибкие и динамические стили.
