# Разница между блочным и строчным (инлайновым) элементами?

В CSS элементы делятся на блочные (block) и строчные (inline) по способу отображения и поведению в потоке документа. Понимание разницы критично для правильной верстки.

## Блочные элементы (Block)

### Характеристики:

- ✅ Занимают всю доступную ширину
- ✅ Начинаются с новой строки
- ✅ Могут иметь width и height
- ✅ Могут иметь margin и padding со всех сторон
- ✅ Могут содержать другие блочные и строчные элементы

### Примеры блочных элементов:

```html
<div>, <p>, <h1>-<h6>, <section>, <article>, 
<header>, <footer>, <nav>, <ul>, <ol>, <li>, 
<table>, <form>, <blockquote>
```

### Поведение:

```html
<div>Блок 1</div>
<div>Блок 2</div>
```

**Результат:**
```
Блок 1
Блок 2
```
(Каждый на новой строке, занимает всю ширину)

## Строчные элементы (Inline)

### Характеристики:

- ✅ Занимают только необходимое пространство
- ✅ Не начинают новую строку
- ✅ width и height не работают
- ✅ Вертикальные margin и padding не работают
- ✅ Могут содержать только текст и другие строчные элементы

### Примеры строчных элементов:

```html
<span>, <a>, <strong>, <em>, <img>, <code>, 
<button>, <input>, <label>, <time>
```

### Поведение:

```html
<span>Строка 1</span>
<span>Строка 2</span>
```

**Результат:**
```
Строка 1 Строка 2
```
(В одной строке, занимают только необходимое место)

## Изменение типа отображения

### `display: block` — сделать блочным:

```css
span {
    display: block; /* Теперь блочный */
}
```

### `display: inline` — сделать строчным:

```css
div {
    display: inline; /* Теперь строчный */
}
```

### `display: inline-block` — гибрид:

```css
.element {
    display: inline-block; /* Строчный, но с возможностями блочного */
}
```

**Характеристики inline-block:**
- Располагается в строке (как inline)
- Может иметь width и height (как block)
- Может иметь margin и padding со всех сторон

## Сравнение

| Характеристика | Block | Inline |
|----------------|-------|--------|
| **Ширина** | Вся доступная | Только контент |
| **Новая строка** | Да | Нет |
| **width/height** | Работают | Не работают |
| **margin/padding** | Все стороны | Только горизонтальные |
| **Содержимое** | Блочные и строчные | Только строчные |

## Практические примеры

### Блочные элементы:

```html
<div class="container">
    <h1>Заголовок</h1>
    <p>Параграф текста</p>
    <div class="box">Блок</div>
</div>
```

```css
.container {
    width: 100%;
    max-width: 1200px;
    margin: 0 auto;
}

.box {
    width: 300px;
    height: 200px;
    margin: 20px;
    padding: 10px;
}
```

### Строчные элементы:

```html
<p>
    Это <strong>жирный</strong> текст с 
    <a href="#">ссылкой</a> и <em>курсивом</em>.
</p>
```

```css
strong {
    color: red;
    /* width и height не работают */
    /* margin-top/bottom не работают */
}

a {
    text-decoration: none;
    padding: 5px 10px; /* Работает */
}
```

### Inline-block для навигации:

```html
<nav>
    <a href="#">Главная</a>
    <a href="#">О нас</a>
    <a href="#">Контакты</a>
</nav>
```

```css
nav a {
    display: inline-block;
    padding: 10px 20px;
    width: 100px; /* Работает! */
    text-align: center;
}
```

## Вложенность

### Правильная вложенность:

```html
<!-- Блочный может содержать строчные -->
<div>
    <p>Текст с <strong>выделением</strong></p>
</div>

<!-- Строчный может содержать только строчные -->
<p>Текст с <strong>выделением</strong> и <a href="#">ссылкой</a></p>
```

### Неправильная вложенность:

```html
<!-- Строчный не может содержать блочный -->
<span><div>Блок</div></span> <!-- ОШИБКА! -->

<!-- Блочный в строчном -->
<p><div>Блок</div></p> <!-- ОШИБКА! -->
```

## Заключение

**Разница:**

- **Блочные элементы:**
  - Занимают всю ширину
  - Начинаются с новой строки
  - Могут иметь width, height, margin, padding
  - Могут содержать блочные и строчные элементы

- **Строчные элементы:**
  - Занимают только необходимое место
  - В одной строке
  - width и height не работают
  - Вертикальные margin/padding не работают
  - Могут содержать только строчные элементы

**Изменение типа:**
- `display: block` — сделать блочным
- `display: inline` — сделать строчным
- `display: inline-block` — гибрид (строчный с возможностями блочного)

Правильное понимание блочных и строчных элементов — основа CSS верстки.
