# Что такое специфичность селектора? Как считать вес селектора?

Специфичность (specificity) — это механизм CSS, который определяет, какое правило стилей будет применено к элементу, когда несколько правил конфликтуют. Специфичность вычисляется как вес селектора.

## Что такое специфичность?

Специфичность — это числовое значение, которое определяет приоритет CSS-правила. Чем выше специфичность, тем выше приоритет.

## Как считать специфичность?

Специфичность считается в формате: `(a, b, c, d)`

Где:
- **a** — инлайновые стили (1000)
- **b** — ID селекторы (100)
- **c** — классы, атрибуты, псевдоклассы (10)
- **d** — теги, псевдоэлементы (1)

## Примеры расчета

### Простые селекторы:

```css
p { }                    /* (0,0,0,1) = 1 */
.class { }               /* (0,0,1,0) = 10 */
#id { }                  /* (0,1,0,0) = 100 */
```

### Комбинированные:

```css
div.container { }        /* (0,0,1,1) = 11 */
#header .nav { }         /* (0,1,1,0) = 110 */
div#main.container { }   /* (0,1,1,1) = 111 */
```

### Сложные:

```css
.container .item:hover { }  /* (0,0,2,0) = 20 */
#header nav a.active { }   /* (0,1,2,2) = 122 */
```

## Порядок приоритета

1. Инлайновые стили: `style="..."` — 1000
2. ID: `#id` — 100
3. Классы: `.class`, `[attr]`, `:pseudo-class` — 10
4. Теги: `div`, `p`, `::pseudo-element` — 1

## Примеры конфликтов

### Пример 1:

```css
p { color: blue; }        /* Специфичность: 1 */
.text { color: red; }     /* Специфичность: 10 */
```

```html
<p class="text">Текст</p>
```

**Результат:** Красный (класс имеет большую специфичность)

### Пример 2:

```css
.container { color: blue; }  /* Специфичность: 10 */
#header { color: red; }      /* Специфичность: 100 */
```

```html
<div id="header" class="container">Текст</div>
```

**Результат:** Красный (ID имеет большую специфичность)

### Пример 3:

```css
div.container { color: blue; }  /* Специфичность: 11 */
.container { color: red; }     /* Специфичность: 10 */
```

```html
<div class="container">Текст</div>
```

**Результат:** Синий (больше специфичность)

## !important

`!important` переопределяет все, кроме другого `!important`:

```css
.text { color: blue !important; }  /* Победит */
#header { color: red; }
```

## Универсальный селектор

`*` имеет специфичность 0:

```css
* { color: blue; }  /* Специфичность: 0 */
```

## Заключение

**Специфичность определяется:**

1. Инлайновые стили — 1000
2. ID — 100
3. Классы, атрибуты, псевдоклассы — 10
4. Теги, псевдоэлементы — 1

**Правила:**
- Больше специфичность = выше приоритет
- При равной специфичности применяется последнее правило
- `!important` переопределяет все

Понимание специфичности помогает контролировать применение стилей и избегать конфликтов.
