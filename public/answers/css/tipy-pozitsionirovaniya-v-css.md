# Типы позиционирования в CSS?

В CSS существует пять типов позиционирования элементов: `static`, `relative`, `absolute`, `fixed` и `sticky`. Каждый тип определяет, как элемент позиционируется на странице и как он взаимодействует с другими элементами.

## Типы позиционирования

### 1. **`static`** — Статическое позиционирование (по умолчанию)

Элементы располагаются в обычном потоке документа.

```css
.element {
    position: static;
}
```

**Характеристики:**
- ✅ Значение по умолчанию
- ✅ Элемент в обычном потоке
- ✅ Свойства `top`, `right`, `bottom`, `left` не работают
- ✅ `z-index` не работает

### 2. **`relative`** — Относительное позиционирование

Элемент позиционируется относительно своего нормального положения.

```css
.element {
    position: relative;
    top: 20px;
    left: 30px;
}
```

**Характеристики:**
- ✅ Элемент остается в потоке
- ✅ Смещается относительно исходной позиции
- ✅ Свойства `top`, `right`, `bottom`, `left` работают
- ✅ Место в потоке сохраняется

**Пример:**
```html
<div class="box box-1">Box 1</div>
<div class="box box-2">Box 2</div>

<style>
.box-2 {
    position: relative;
    top: -20px;
    left: 20px;
}
</style>
```

### 3. **`absolute`** — Абсолютное позиционирование

Элемент позиционируется относительно ближайшего позиционированного родителя.

```css
.element {
    position: absolute;
    top: 50px;
    right: 20px;
}
```

**Характеристики:**
- ✅ Удаляется из потока документа
- ✅ Позиционируется относительно ближайшего родителя с `position: relative/absolute/fixed`
- ✅ Если такого родителя нет — относительно `<body>`
- ✅ Не занимает место в потоке

**Пример:**
```html
<div class="container">
    <div class="absolute-box">Абсолютный</div>
</div>

<style>
.container {
    position: relative; /* Контекст позиционирования */
}

.absolute-box {
    position: absolute;
    top: 10px;
    right: 10px;
}
</style>
```

### 4. **`fixed`** — Фиксированное позиционирование

Элемент позиционируется относительно окна браузера.

```css
.element {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
}
```

**Характеристики:**
- ✅ Удаляется из потока
- ✅ Позиционируется относительно viewport
- ✅ Остается на месте при прокрутке
- ✅ Не занимает место в потоке

**Пример:**
```css
.header {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    background: white;
    z-index: 1000;
}
```

### 5. **`sticky`** — Липкое позиционирование

Элемент ведет себя как `relative`, пока не достигнет указанной позиции, затем становится `fixed`.

```css
.element {
    position: sticky;
    top: 0;
}
```

**Характеристики:**
- ✅ Гибрид `relative` и `fixed`
- ✅ Остается в потоке до достижения порога
- ✅ Затем "прилипает" к указанной позиции
- ✅ Работает только с одним свойством (`top`, `bottom`, `left`, `right`)

**Пример:**
```css
.sticky-header {
    position: sticky;
    top: 0;
    background: white;
    z-index: 100;
}
```

## Свойства позиционирования

### `top`, `right`, `bottom`, `left`

Определяют смещение элемента:

```css
.element {
    position: absolute;
    top: 20px;      /* Отступ сверху */
    right: 30px;    /* Отступ справа */
    bottom: 40px;   /* Отступ снизу */
    left: 50px;     /* Отступ слева */
}
```

### `z-index`

Определяет порядок наложения:

```css
.element-1 {
    position: absolute;
    z-index: 1;
}

.element-2 {
    position: absolute;
    z-index: 2; /* Будет поверх element-1 */
}
```

## Практические примеры

### Центрирование абсолютного элемента:

```css
.center {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}
```

### Фиксированная навигация:

```css
.nav {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    background: #333;
    z-index: 1000;
}
```

### Липкий заголовок таблицы:

```css
table th {
    position: sticky;
    top: 0;
    background: white;
}
```

## Заключение

**Типы позиционирования:**

1. **`static`** — обычный поток (по умолчанию)
2. **`relative`** — относительно исходной позиции
3. **`absolute`** — относительно родителя
4. **`fixed`** — относительно viewport
5. **`sticky`** — липкое позиционирование

**Использование:**
- `static` — по умолчанию, для обычного потока
- `relative` — для смещения элементов
- `absolute` — для точного позиционирования
- `fixed` — для фиксированных элементов (навигация)
- `sticky` — для липких элементов (заголовки таблиц)

Правильный выбор типа позиционирования критичен для создания правильных макетов и взаимодействия элементов.
