# Глобальные ключевые слова в CSS?

Глобальные ключевые слова в CSS — это значения, которые можно использовать для любого CSS-свойства. Они обеспечивают единообразное поведение и упрощают работу со стилями.

## Глобальные ключевые слова

### 1. **`initial` — Начальное значение**

```css
.element {
    color: initial; /* Значение по умолчанию для свойства */
}
```

Возвращает свойство к его начальному значению.

### 2. **`inherit` — Наследование**

```css
.element {
    color: inherit; /* Наследует от родителя */
}
```

Элемент наследует значение от родительского элемента.

### 3. **`unset` — Сброс**

```css
.element {
    color: unset; /* Наследует, если можно, иначе initial */
}
```

Если свойство наследуемое — наследует, иначе — `initial`.

### 4. **`revert` — Откат**

```css
.element {
    color: revert; /* Откатывает к значению из таблицы стилей браузера */
}
```

Откатывает к значению из таблицы стилей браузера или каскада.

### 5. **`revert-layer` — Откат слоя**

```css
.element {
    color: revert-layer; /* Откатывает к значению из предыдущего слоя */
}
```

Откатывает к значению из предыдущего слоя каскада.

## Примеры использования

### Сброс стилей:

```css
.button {
    color: initial; /* Сброс к начальному значению */
    background: initial;
}
```

### Наследование:

```css
.child {
    color: inherit; /* Наследует цвет от родителя */
}
```

### Условный сброс:

```css
.element {
    color: unset; /* Наследует или initial */
}
```
