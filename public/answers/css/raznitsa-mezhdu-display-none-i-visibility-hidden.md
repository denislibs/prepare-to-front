# Разница между `display: none` и `visibility: hidden`?

`display: none` и `visibility: hidden` — это два способа скрыть элементы в CSS, но они работают по-разному и имеют разные последствия для макета и доступности.

## Разница между `display: none` и `visibility: hidden`

### **`display: none`** — Полное удаление из потока

Элемент полностью удаляется из потока документа.

```css
.element {
    display: none;
}
```

**Характеристики:**
- ✅ Элемент не отображается
- ✅ Элемент не занимает место в макете
- ✅ Не влияет на расположение других элементов
- ✅ Не доступен для скринридеров (обычно)
- ✅ Не реагирует на события мыши

### **`visibility: hidden`** — Скрытие с сохранением места

Элемент скрыт, но занимает место в макете.

```css
.element {
    visibility: hidden;
}
```

**Характеристики:**
- ✅ Элемент не отображается
- ✅ Элемент занимает место в макете
- ✅ Влияет на расположение других элементов
- ✅ Доступен для скринридеров (может быть)
- ✅ Не реагирует на события мыши

## Визуальное сравнение

### С `display: none`:

```html
<div>Элемент 1</div>
<div style="display: none;">Скрытый</div>
<div>Элемент 3</div>
```

**Результат:**
```
Элемент 1
Элемент 3
```
(Скрытый элемент не занимает места)

### С `visibility: hidden`:

```html
<div>Элемент 1</div>
<div style="visibility: hidden;">Скрытый</div>
<div>Элемент 3</div>
```

**Результат:**
```
Элемент 1
[пустое место]
Элемент 3
```
(Скрытый элемент занимает место)

## Практические примеры

### Модальное окно:

```css
.modal {
    display: none; /* Полностью скрыто, не занимает места */
}

.modal.active {
    display: block; /* Показывается при активации */
}
```

### Плавное появление:

```css
.element {
    visibility: hidden; /* Скрыто, но место сохранено */
    opacity: 0;
    transition: opacity 0.3s;
}

.element.visible {
    visibility: visible;
    opacity: 1;
}
```

### Скрытие при загрузке:

```css
.loading {
    display: none; /* Не занимает места */
}

.loading.active {
    display: block;
}
```

## Влияние на макет

### `display: none` — перестройка макета:

```css
.box {
    display: none;
}

/* Другие элементы сдвигаются, занимая освободившееся место */
```

### `visibility: hidden` — макет сохраняется:

```css
.box {
    visibility: hidden;
}

/* Другие элементы остаются на месте, место для скрытого элемента сохраняется */
```

## Доступность

### `display: none`:

- Обычно не доступен для скринридеров
- Полностью удален из DOM дерева (визуально)

### `visibility: hidden`:

- Может быть доступен для скринридеров (зависит от реализации)
- Остается в DOM дереве

## Анимации и переходы

### `display: none` — нельзя анимировать:

```css
/* НЕ РАБОТАЕТ */
.element {
    display: none;
    transition: display 0.3s; /* Не работает */
}
```

### `visibility: hidden` — можно анимировать:

```css
.element {
    visibility: hidden;
    opacity: 0;
    transition: visibility 0.3s, opacity 0.3s;
}

.element.visible {
    visibility: visible;
    opacity: 1;
}
```

## Комбинация с другими свойствами

### С `opacity`:

```css
/* display: none + opacity не имеет смысла */
.element {
    display: none;
    opacity: 0; /* Не видно, так как display: none */
}

/* visibility: hidden + opacity работает */
.element {
    visibility: hidden;
    opacity: 0; /* Можно анимировать */
}
```

## Когда использовать что?

### Используйте `display: none` когда:

- ✅ Элемент не нужен в макете
- ✅ Нужно полностью скрыть элемент
- ✅ Элемент не должен занимать место
- ✅ Модальные окна, выпадающие меню

**Примеры:**
- Модальные окна
- Выпадающие меню
- Скрытые секции
- Элементы, которые показываются условно

### Используйте `visibility: hidden` когда:

- ✅ Нужно сохранить место в макете
- ✅ Нужна анимация появления/исчезновения
- ✅ Элемент временно скрыт, но место нужно сохранить

**Примеры:**
- Плавное появление элементов
- Сохранение макета при скрытии
- Анимации показа/скрытия

## Альтернативы

### `opacity: 0`:

```css
.element {
    opacity: 0; /* Невидим, но занимает место и реагирует на события */
    pointer-events: none; /* Отключить события мыши */
}
```

### `height: 0` + `overflow: hidden`:

```css
.element {
    height: 0;
    overflow: hidden;
    transition: height 0.3s;
}
```

## Заключение

**Разница:**

| Характеристика | `display: none` | `visibility: hidden` |
|----------------|-----------------|----------------------|
| **Отображение** | Не виден | Не виден |
| **Место в макете** | Не занимает | Занимает |
| **Влияние на макет** | Перестройка | Сохранение |
| **Анимация** | Нельзя | Можно |
| **Скринридеры** | Обычно недоступен | Может быть доступен |
| **События мыши** | Не реагирует | Не реагирует |

**Использование:**
- **`display: none`** — полное скрытие, удаление из макета
- **`visibility: hidden`** — скрытие с сохранением места

**Рекомендации:**
- Используйте `display: none` для полного скрытия
- Используйте `visibility: hidden` для сохранения макета
- Комбинируйте с `opacity` для плавных анимаций

Правильный выбор зависит от задачи и требований к макету.
