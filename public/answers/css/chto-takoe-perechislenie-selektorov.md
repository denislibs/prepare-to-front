# Что такое перечисление селекторов?

Перечисление селекторов (selector list) в CSS — это способ применения одних и тех же стилей к нескольким элементам, перечисляя селекторы через запятую. Это упрощает код и улучшает поддерживаемость.

## Что такое перечисление селекторов?

Перечисление селекторов — это список селекторов, разделенных запятыми, к которым применяются одни и те же стили.

## Использование

### Базовый синтаксис:

```css
селектор1, селектор2, селектор3 {
    /* Общие стили */
}
```

## Примеры использования

### Несколько элементов:

```css
h1, h2, h3 {
    color: blue;
    font-weight: bold;
}
```

### Классы и элементы:

```css
.button, .link, a {
    color: #3498db;
    text-decoration: none;
}
```

### Комбинированные селекторы:

```css
.header h1, .header h2, .header h3 {
    margin-top: 0;
}
```

### С псевдоклассами:

```css
a:hover, .button:hover, .link:hover {
    color: red;
}
```

## Преимущества

### 1. **Упрощение кода**

```css
/* Вместо */
h1 { color: blue; }
h2 { color: blue; }
h3 { color: blue; }

/* Можно написать */
h1, h2, h3 {
    color: blue;
}
```

### 2. **Легче поддерживать**

Изменения в одном месте применяются ко всем элементам.

### 3. **Меньше дублирования**

Один блок стилей вместо нескольких.

## Лучшие практики

### Группировка логически связанных элементов:

```css
/* Хорошо */
.button, .link {
    color: blue;
}

/* Плохо */
.button, .card, .modal, .tooltip {
    color: blue; /* Не связанные элементы */
}
```

### Читаемость:

```css
/* Хорошо */
h1, h2, h3 {
    color: blue;
}

/* Плохо */
h1,h2,h3{color:blue;} /* Сложно читать */
```

1. ✅ Применяет одни стили к нескольким элементам
2. ✅ Упрощает код

**Использование:**

- Группировка связанных элементов
- Упрощение кода

**Рекомендации:**

- Сохраняйте читаемость
