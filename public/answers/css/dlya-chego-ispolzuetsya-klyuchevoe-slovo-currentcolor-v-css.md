# Для чего используется ключевое слово `currentColor` в CSS?

Ключевое слово `currentColor` в CSS позволяет использовать текущее значение свойства `color` элемента для других свойств, которые принимают цвет. Это упрощает поддержку и создает более гибкие стили.

## Что такое `currentColor`?

`currentColor` — это ключевое слово, которое представляет текущее значение свойства `color` элемента. Оно может использоваться везде, где ожидается цвет.

## Использование

### Базовый синтаксис:

```css
.element {
    color: blue;
    border: 2px solid currentColor; /* Использует цвет текста */
}
```

## Примеры использования

### Границы:

```css
.button {
    color: blue;
    border: 2px solid currentColor; /* Граница того же цвета, что и текст */
}
```

### Тени:

```css
.element {
    color: red;
    box-shadow: 0 2px 4px currentColor; /* Тень того же цвета */
}
```

### Псевдоэлементы:

```css
.link {
    color: blue;
}

.link::after {
    content: "→";
    color: currentColor; /* Наследует цвет от .link */
}
```

### SVG:

```css
.icon {
    color: green;
    fill: currentColor; /* SVG использует цвет текста */
}
```

## Преимущества

### 1. **Консистентность**

Цвета автоматически синхронизируются.

### 2. **Упрощение поддержки**

Изменение одного цвета обновляет все связанные элементы.

### 3. **Гибкость**

Легко создавать темы и вариации.

## Практические примеры

### Кнопка:

```css
.button {
    color: blue;
    border: 2px solid currentColor;
    background-color: transparent;
}

.button:hover {
    color: darkblue; /* Граница автоматически обновится */
}
```

### Иконка:

```css
.icon {
    color: #333;
    fill: currentColor; /* SVG иконка использует цвет текста */
}
```

### Подчеркивание:

```css
.link {
    color: blue;
    text-decoration: none;
    border-bottom: 1px solid currentColor; /* Подчеркивание того же цвета */
}
```

## Заключение

**`currentColor` используется для:**

1. ✅ Использования цвета текста в других свойствах
2. ✅ Создания консистентных стилей
3. ✅ Упрощения поддержки

**Использование:**

- Границы
- Тени
- SVG fill/stroke
- Псевдоэлементы

**Рекомендации:**

- Используйте для синхронизации цветов
- Комбинируйте с CSS переменными
- Упрощайте поддержку тем

Правильное использование `currentColor` создает более гибкие и поддерживаемые стили.
