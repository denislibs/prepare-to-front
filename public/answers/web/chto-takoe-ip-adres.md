# Что такое IP-адрес?

IP-адрес (Internet Protocol address) — это уникальный числовой идентификатор устройства в сети. IP-адрес используется для идентификации и маршрутизации данных в сети Интернет.

## Что такое IP-адрес?

IP-адрес — это адрес устройства в сети, который позволяет другим устройствам находить и обмениваться данными с этим устройством.

## Типы IP-адресов

### 1. **IPv4**

32-битный адрес, записанный в виде четырех чисел.

```
192.168.1.1
```

### 2. **IPv6**

128-битный адрес, записанный в шестнадцатеричном формате.

```
2001:0db8:85a3:0000:0000:8a2e:0370:7334
```

## Классы IP-адресов

### 1. **Публичный IP**

Доступен из Интернета.

### 2. **Приватный IP**

Используется в локальной сети.

```
192.168.x.x
10.x.x.x
172.16.x.x - 172.31.x.x
```

## DNS и IP-адреса

DNS преобразует доменные имена в IP-адреса.

```
example.com → 93.184.216.34
```
