# Что такое `HTTP/3`? Особенности `HTTP/3`?

HTTP/3 — это новая версия протокола HTTP, которая использует QUIC (Quick UDP Internet Connections) вместо TCP. HTTP/3 предлагает улучшенную производительность, особенно при нестабильных соединениях.

## Что такое HTTP/3?

HTTP/3 — это третья версия протокола HTTP, которая использует QUIC поверх UDP вместо TCP, что обеспечивает более быструю и надежную передачу данных.

## Особенности

### 1. **QUIC вместо TCP**

Использует UDP с встроенной надежностью.

```
HTTP/1, HTTP/2: HTTP → TCP → IP
HTTP/3: HTTP → QUIC → UDP → IP
```

### 2. **Встроенное шифрование**

Шифрование на уровне протокола.

### 3. **Мультиплексирование без блокировки**

Несколько потоков без блокировки друг друга.

### 4. **Быстрое восстановление соединения**

Быстрое восстановление при потере пакетов.

### 5. **Улучшенная производительность**

Лучшая производительность на нестабильных соединениях.

## Преимущества

### 1. **Быстрее установка соединения**

0-RTT для повторных соединений.

### 2. **Лучше на мобильных**

Оптимизирован для мобильных сетей.

### 3. **Меньше задержек**

Снижение задержек передачи данных.
