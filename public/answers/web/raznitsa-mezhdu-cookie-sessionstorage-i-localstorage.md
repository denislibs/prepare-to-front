# Разница между `cookie`, `sessionStorage` и `localStorage`?

Cookies, sessionStorage и localStorage — это три разных способа хранения данных в браузере. Понимание различий помогает выбрать правильный метод для конкретной задачи.

## Cookies

### Характеристики:

- Отправляются с каждым HTTP-запросом
- Ограничение размера: ~4 КБ
- Могут иметь срок действия
- Доступны на сервере

```javascript
// Установка cookie
document.cookie = "name=value; expires=Thu, 18 Dec 2024 12:00:00 UTC; path=/";

// Чтение cookie
const cookies = document.cookie;
```

## sessionStorage

### Характеристики:

- Хранится только в рамках сеанса
- Удаляется при закрытии вкладки
- Объем: ~5-10 МБ
- Доступно только через JavaScript

```javascript
// Сохранение
sessionStorage.setItem('key', 'value');

// Получение
const value = sessionStorage.getItem('key');
```

## localStorage

### Характеристики:

- Хранится постоянно
- Не удаляется при закрытии браузера
- Объем: ~5-10 МБ
- Доступно только через JavaScript

```javascript
// Сохранение
localStorage.setItem('key', 'value');

// Получение
const value = localStorage.getItem('key');
```

## Сравнение

| Характеристика | Cookie | sessionStorage | localStorage |
|----------------|--------|----------------|--------------|
| Размер | ~4 КБ | ~5-10 МБ | ~5-10 МБ |
| Срок действия | Настраивается | До закрытия вкладки | Постоянно |
| Доступ на сервере | ✅ Да | ❌ Нет | ❌ Нет |
| Отправка с запросами | ✅ Да | ❌ Нет | ❌ Нет |

## Когда использовать

### Cookies:

- ✅ Аутентификация
- ✅ Данные для сервера
- ✅ Небольшие данные

### sessionStorage:

- ✅ Временные данные сеанса
- ✅ Данные формы
- ✅ Временные настройки

### localStorage:

- ✅ Настройки пользователя
- ✅ Кэш данных
- ✅ Постоянные данные

## Заключение

**Cookies:**

1. ✅ Отправляются с запросами
2. ✅ Доступны на сервере
3. ❌ Ограниченный размер

**sessionStorage:**

1. ✅ Только для сеанса
2. ✅ Больше объема
3. ❌ Удаляется при закрытии

**localStorage:**

1. ✅ Постоянное хранение
2. ✅ Больше объема
3. ❌ Только на клиенте

**Рекомендации:**

- Используйте cookies для серверных данных
- Используйте sessionStorage для временных данных
- Используйте localStorage для постоянных данных

Правильный выбор зависит от требований к данным.
