# Что такое CDN?

CDN (Content Delivery Network) — это сеть распределенных серверов, которые доставляют контент пользователям с ближайшего к ним сервера. CDN улучшает производительность и доступность веб-сайтов.

## Что такое CDN?

CDN — это сеть серверов, расположенных в разных географических точках, которые кэшируют и доставляют контент пользователям с минимальной задержкой.

## Как это работает

### 1. **Распределение серверов**

Серверы расположены в разных регионах.

```
Пользователь → Ближайший CDN сервер → Контент
```

### 2. **Кэширование**

Контент кэшируется на серверах CDN.

### 3. **Быстрая доставка**

Контент доставляется с ближайшего сервера.

## Преимущества

### 1. **Производительность**

Быстрая загрузка контента.

### 2. **Доступность**

Высокая доступность даже при нагрузке.

### 3. **Масштабируемость**

Легко масштабируется под нагрузку.

### 4. **Экономия трафика**

Снижение нагрузки на основной сервер.

## Примеры CDN

### 1. **Cloudflare**

Популярный CDN с дополнительными функциями безопасности.

### 2. **Amazon CloudFront**

CDN от Amazon Web Services.

### 3. **Google Cloud CDN**

CDN от Google Cloud Platform.

## Использование

```html
<!-- Подключение библиотеки через CDN -->
<script src="https://cdn.jsdelivr.net/npm/jquery@3.6.0/dist/jquery.min.js"></script>
```
