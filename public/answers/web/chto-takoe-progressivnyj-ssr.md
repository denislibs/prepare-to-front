# Что такое прогрессивный SSR?

Прогрессивный SSR (Progressive Server-Side Rendering) — это гибридный подход к рендерингу, который сочетает серверный рендеринг для первоначальной загрузки с клиентским рендерингом для последующих обновлений. Это обеспечивает быструю первоначальную загрузку и интерактивность.

## Что такое прогрессивный SSR?

Прогрессивный SSR — это техника, при которой страница рендерится на сервере для первоначальной загрузки, а затем "гидратируется" на клиенте для интерактивности.

## Принципы

### 1. **Серверный рендеринг**

Первоначальный HTML генерируется на сервере.

```javascript
// На сервере
const html = ReactDOMServer.renderToString(<App />);
```

### 2. **Гидратация на клиенте**

React "оживляет" статический HTML на клиенте.

```javascript
// На клиенте
ReactDOM.hydrateRoot(document.getElementById('root'), <App />);
```

### 3. **Постепенный переход**

Переход от SSR к CSR для последующих обновлений.

## Преимущества

### 1. **Быстрая первоначальная загрузка**

Пользователь видит контент сразу.

### 2. **SEO-оптимизация**

Поисковые системы индексируют полный HTML.

### 3. **Интерактивность**

После гидратации страница становится полностью интерактивной.

## Примеры

### Next.js (прогрессивный SSR):

```jsx
// Страница с SSR
export default function Page({ data }) {
    return <div>{data.title}</div>;
}

// Данные загружаются на сервере
export async function getServerSideProps() {
    const data = await fetchData();
    return { props: { data } };
}
```

### Постепенная гидратация:

```jsx
// Критический контент рендерится на сервере
function App() {
    return (
        <div>
            <Header /> {/* SSR */}
            <MainContent /> {/* SSR */}
            <LazyComponent /> {/* CSR после гидратации */}
        </div>
    );
}
```

## Заключение

**Прогрессивный SSR — это:**

1. ✅ Гибридный подход к рендерингу
2. ✅ SSR для первоначальной загрузки
3. ✅ CSR для интерактивности

**Преимущества:**

- Быстрая первоначальная загрузка
- SEO-оптимизация
- Интерактивность

**Рекомендации:**

- Используйте для критического контента
- Оптимизируйте гидратацию
- Комбинируйте с ленивой загрузкой

Прогрессивный SSR — мощный подход для создания быстрых и SEO-дружелюбных приложений.
