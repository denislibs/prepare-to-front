# Что такое OSI модель?

OSI модель (Open Systems Interconnection) — это концептуальная модель, которая описывает семь уровней сетевого взаимодействия. Она используется для понимания того, как данные передаются по сети от одного устройства к другому.

## Что такое OSI модель?

OSI модель — это эталонная модель взаимодействия открытых систем, которая разделяет процесс сетевой коммуникации на семь уровней, каждый из которых выполняет свою функцию.

## Семь уровней OSI

### 1. **Физический уровень (Physical Layer)**

Передача битов по физической среде.

- Кабели, провода
- Радиосигналы
- Электрические сигналы

### 2. **Канальный уровень (Data Link Layer)**

Обеспечение надежной передачи данных между соседними узлами.

- MAC-адреса
- Обработка ошибок
- Контроль доступа к среде

### 3. **Сетевой уровень (Network Layer)**

Маршрутизация пакетов через сеть.

- IP-адреса
- Маршрутизация
- Логическая адресация

### 4. **Транспортный уровень (Transport Layer)**

Обеспечение надежной передачи данных между конечными точками.

- TCP, UDP
- Контроль ошибок
- Управление потоком

### 5. **Сеансовый уровень (Session Layer)**

Управление сеансами связи.

- Установка сеанса
- Управление диалогом
- Синхронизация

### 6. **Уровень представления (Presentation Layer)**

Преобразование данных в формат, понятный приложению.

- Шифрование
- Сжатие
- Кодирование

### 7. **Прикладной уровень (Application Layer)**

Интерфейс для взаимодействия с приложениями.

- HTTP, HTTPS
- FTP, SMTP
- DNS

## Пример передачи данных

### От приложения к сети:

```
Приложение → HTTP → TCP → IP → Ethernet → Физическая среда
```

### От сети к приложению:

```
Физическая среда → Ethernet → IP → TCP → HTTP → Приложение
```
