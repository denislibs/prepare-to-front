# Что такое `HTTP`?

HTTP (HyperText Transfer Protocol) — это протокол прикладного уровня для передачи данных в сети Интернет. HTTP является основой веб-коммуникации и используется для загрузки веб-страниц и обмена данными между клиентом и сервером.

## Что такое HTTP?

HTTP — это протокол, который определяет формат и правила обмена данными между клиентом (браузером) и сервером. HTTP работает поверх TCP/IP и использует модель запрос-ответ.

## Основные характеристики

### 1. **Протокол прикладного уровня**

Работает поверх TCP/IP.

```
HTTP → TCP → IP → Физический уровень
```

### 2. **Модель запрос-ответ**

Клиент отправляет запрос, сервер отвечает.

```
Клиент → Запрос → Сервер
Клиент ← Ответ ← Сервер
```

### 3. **Без состояния (Stateless)**

Каждый запрос независим.

### 4. **Текстовый протокол**

Человекочитаемый формат.

## Структура запроса

```
GET /index.html HTTP/1.1
Host: example.com
User-Agent: Mozilla/5.0
Accept: text/html
```

## Структура ответа

```
HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1234

<html>...</html>
```

## Методы

- GET — получение данных
- POST — отправка данных
- PUT — обновление данных
- DELETE — удаление данных
- HEAD — получение заголовков
- OPTIONS — информация о методах

## Статус-коды

- 200 OK — успешно
- 404 Not Found — не найдено
- 500 Server Error — ошибка сервера
