# Что такое Progressive Web Application (PWA)?

Progressive Web Application (PWA) — это веб-приложение, которое использует современные веб-технологии для обеспечения нативного опыта использования, как у мобильных приложений. PWA работают в браузере, но могут быть установлены на устройство и работать офлайн.

## Что такое PWA?

PWA — это веб-приложение, которое сочетает лучшие качества веб-сайтов и нативных приложений, обеспечивая быструю загрузку, работу офлайн и нативный пользовательский опыт.

## Основные характеристики

### 1. **Работает офлайн**

Service Workers позволяют работать без интернета.

```javascript
// service-worker.js
self.addEventListener('install', (event) => {
    event.waitUntil(
        caches.open('v1').then((cache) => {
            return cache.addAll([
                '/',
                '/index.html',
                '/styles.css',
                '/app.js'
            ]);
        })
    );
});
```

### 2. **Установка на устройство**

Можно установить как нативное приложение.

```json
// manifest.json
{
    "name": "Мое приложение",
    "short_name": "Приложение",
    "start_url": "/",
    "display": "standalone",
    "background_color": "#ffffff",
    "theme_color": "#000000",
    "icons": [
        {
            "src": "/icon-192.png",
            "sizes": "192x192",
            "type": "image/png"
        }
    ]
}
```

### 3. **Быстрая загрузка**

Оптимизация производительности и кэширование.

### 4. **Адаптивный дизайн**

Работает на всех устройствах.

## Технологии

### 1. **Service Workers**

Для офлайн-функциональности и кэширования.

```javascript
// Регистрация Service Worker
if ('serviceWorker' in navigator) {
    navigator.serviceWorker.register('/sw.js');
}
```

### 2. **Web App Manifest**

Метаданные для установки приложения.

```html
<link rel="manifest" href="/manifest.json">
```

### 3. **HTTPS**

Обязательное требование для PWA.

### 4. **Responsive Design**

Адаптивный дизайн для всех устройств.

## Преимущества

### 1. **Нативный опыт**

Ощущение как у нативного приложения.

### 2. **Офлайн-работа**

Работает без интернета.

### 3. **Быстрая загрузка**

Оптимизированная производительность.

### 4. **Обновления**

Автоматические обновления без установки.

### 5. **Кроссплатформенность**

Работает на всех платформах.

## Примеры

### Базовое PWA:

```html
<!DOCTYPE html>
<html>
<head>
    <link rel="manifest" href="/manifest.json">
    <meta name="theme-color" content="#000000">
</head>
<body>
    <h1>PWA Приложение</h1>
    <script>
        if ('serviceWorker' in navigator) {
            navigator.serviceWorker.register('/sw.js');
        }
    </script>
</body>
</html>
```
