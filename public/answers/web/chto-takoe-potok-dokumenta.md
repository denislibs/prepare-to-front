# Что такое поток документа?

Поток документа (Document Flow) — это порядок, в котором элементы HTML отображаются на странице. Понимание потока документа критично для работы с CSS и позиционированием элементов.

## Что такое поток документа?

Поток документа — это естественный порядок отображения элементов HTML сверху вниз и слева направо, в том порядке, в котором они появляются в HTML-коде.

## Нормальный поток

### Блочные элементы:

```html
<div>Блок 1</div>
<div>Блок 2</div>
<div>Блок 3</div>
```

Отображаются друг под другом.

### Строчные элементы:

```html
<span>Текст 1</span>
<span>Текст 2</span>
<span>Текст 3</span>
```

Отображаются друг за другом в одной строке.

## Изменение потока

### 1. **Позиционирование**

```css
/* Абсолютное позиционирование выводит из потока */
.element {
    position: absolute;
    top: 0;
    left: 0;
}
```

### 2. **Float**

```css
/* Float выводит из нормального потока */
.element {
    float: left;
}
```

### 3. **Flexbox**

```css
/* Flexbox изменяет направление потока */
.container {
    display: flex;
    flex-direction: column;
}
```

### 4. **Grid**

```css
/* Grid создает новый поток */
.container {
    display: grid;
    grid-template-columns: 1fr 1fr;
}
```

## Заключение

**Поток документа — это:**

1. ✅ Естественный порядок отображения элементов
2. ✅ Сверху вниз, слева направо
3. ✅ Может быть изменен через CSS

**Изменение потока:**

- Позиционирование
- Float
- Flexbox
- Grid

**Рекомендации:**

- Понимайте нормальный поток
- Используйте CSS для изменения потока
- Тестируйте на разных устройствах
