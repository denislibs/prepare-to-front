# Разница между URI и URL?

URI (Uniform Resource Identifier) и URL (Uniform Resource Locator) — это связанные понятия для идентификации ресурсов в сети. Понимание различий важно для работы с веб-технологиями.

## URI (Uniform Resource Identifier)

### Что это?

URI — это общий термин для идентификации ресурса.

```
URI включает:
- URL (местоположение)
- URN (имя)
```

## URL (Uniform Resource Locator)

### Что это?

URL — это тип URI, который указывает местоположение ресурса.

```
https://example.com/path/to/resource?param=value#fragment
```

## Структура URL

### Компоненты:

```
https://example.com:443/path/to/resource?param=value#fragment
│    │              │   │                  │         │
│    │              │   │                  │         └─ Fragment
│    │              │   │                  └─ Query
│    │              │   └─ Path
│    │              └─ Port
│    └─ Host
└─ Protocol
```

## Различия

### 1. **URI — общий термин**

URI включает URL и URN.

### 2. **URL — конкретное местоположение**

URL указывает, где находится ресурс.

### 3. **URN — имя ресурса**

URN идентифицирует ресурс по имени, а не по местоположению.

## Примеры

### URL:

```
https://example.com/page.html
http://example.com:8080/api/data
```

### URN:

```
urn:isbn:0451450523
urn:uuid:6e8bc430-9c3a-11d9-9669-0800200c9a66
```
