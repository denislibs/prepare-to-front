# Типы операторов в JavaScript?

В JavaScript существует множество типов операторов для выполнения различных операций. Понимание типов операторов критически важно для написания правильного и эффективного кода.

## Типы операторов

### 1. **Арифметические операторы**

Операторы для математических операций.

```javascript
+ // Сложение
- // Вычитание
* // Умножение
/ // Деление
% // Остаток от деления
** // Возведение в степень
++ // Инкремент
-- // Декремент
```

### 2. **Операторы присваивания**

Операторы для присваивания значений.

```javascript
= // Присваивание
+= // Сложение с присваиванием
-= // Вычитание с присваиванием
*= // Умножение с присваиванием
/= // Деление с присваиванием
%= // Остаток с присваиванием
```

### 3. **Операторы сравнения**

Операторы для сравнения значений.

```javascript
== // Равенство (нестрогое)
=== // Равенство (строгое)
!= // Неравенство (нестрогое)
!== // Неравенство (строгое)
> // Больше
< // Меньше
>= // Больше или равно
<= // Меньше или равно
```

### 4. **Логические операторы**

Операторы для логических операций.

```javascript
&& // И
|| // ИЛИ
! // НЕ
?? // Нулевое слияние
```

### 5. **Побитовые операторы**

Операторы для работы с битами.

```javascript
& // Побитовое И
| // Побитовое ИЛИ
^ // Побитовое исключающее ИЛИ
~ // Побитовое НЕ
<< // Сдвиг влево
>> // Сдвиг вправо
>>> // Беззнаковый сдвиг вправо
```

### 6. **Операторы доступа**

Операторы для доступа к свойствам и элементам.

```javascript
. // Доступ к свойству
[] // Доступ к элементу
?. // Опциональная цепочка
```

### 7. **Операторы typeof и instanceof**

Операторы для проверки типов.

```javascript
typeof // Тип значения
instanceof // Принадлежность к классу
```

### 8. **Тернарный оператор**

Условный оператор.

```javascript
condition ? value1 : value2
```

## Практические примеры

### Пример 1: Арифметические операторы

```javascript
const a = 10;
const b = 3;

a + b; // 13
a - b; // 7
a * b; // 30
a / b; // 3.333...
a % b; // 1
a ** b; // 1000
```

### Пример 2: Операторы присваивания

```javascript
let x = 10;
x += 5; // x = 15
x -= 3; // x = 12
x *= 2; // x = 24
x /= 4; // x = 6
```

### Пример 3: Логические операторы

```javascript
true && false; // false
true || false; // true
!true; // false
null ?? "default"; // "default"
```

## Заключение

Типы операторов в JavaScript:

- **Арифметические** — для математических операций
- **Присваивания** — для присваивания значений
- **Сравнения** — для сравнения значений
- **Логические** — для логических операций
- **Побитовые** — для работы с битами
- **Доступа** — для доступа к свойствам
- **Типов** — для проверки типов
- **Тернарный** — для условных операций

**Помните:** понимание типов операторов критически важно для написания правильного JavaScript кода. Каждый тип операторов имеет свое назначение и особенности использования.
