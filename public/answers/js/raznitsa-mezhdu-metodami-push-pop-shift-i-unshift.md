# Разница между методами `.push()`, `.pop()`, `.shift()` и `.unshift()`?

Методы `.push()`, `.pop()`, `.shift()` и `.unshift()` — это методы массивов в JavaScript для добавления и удаления элементов. Они работают с разными концами массива и имеют разные направления операций. Понимание различий критически важно для правильной работы с массивами.

## .push()

`.push()` — добавляет один или несколько элементов в конец массива и возвращает новую длину массива.

### Характеристики:

- ✅ **Добавляет в конец** — элементы добавляются в конец массива
- ✅ **Изменяет исходный массив** — мутирует массив
- ✅ **Возвращает длину** — новую длину массива
- ✅ **Множественные элементы** — можно добавить несколько элементов

### Синтаксис:

```javascript
array.push(element1, element2, ..., elementN)
```

### Примеры:

```javascript
const arr = [1, 2, 3];

arr.push(4); // Добавить один элемент
console.log(arr); // [1, 2, 3, 4]
console.log(arr.push(5)); // 5 (новая длина)

arr.push(6, 7, 8); // Добавить несколько элементов
console.log(arr); // [1, 2, 3, 4, 5, 6, 7, 8]
```

## .pop()

`.pop()` — удаляет последний элемент из массива и возвращает его.

### Характеристики:

- ✅ **Удаляет с конца** — удаляет последний элемент
- ✅ **Изменяет исходный массив** — мутирует массив
- ✅ **Возвращает элемент** — удаленный элемент
- ✅ **Один элемент** — удаляет только один элемент

### Синтаксис:

```javascript
array.pop()
```

### Примеры:

```javascript
const arr = [1, 2, 3, 4];

const last = arr.pop(); // Удалить последний элемент
console.log(last); // 4 (удаленный элемент)
console.log(arr); // [1, 2, 3]

const empty = [];
const result = empty.pop();
console.log(result); // undefined (массив пуст)
```

## .shift()

`.shift()` — удаляет первый элемент из массива и возвращает его.

### Характеристики:

- ✅ **Удаляет с начала** — удаляет первый элемент
- ✅ **Изменяет исходный массив** — мутирует массив
- ✅ **Возвращает элемент** — удаленный элемент
- ⚠️ **Медленнее** — чем pop (нужно сдвигать все элементы)

### Синтаксис:

```javascript
array.shift()
```

### Примеры:

```javascript
const arr = [1, 2, 3, 4];

const first = arr.shift(); // Удалить первый элемент
console.log(first); // 1 (удаленный элемент)
console.log(arr); // [2, 3, 4]

const empty = [];
const result = empty.shift();
console.log(result); // undefined (массив пуст)
```

## .unshift()

`.unshift()` — добавляет один или несколько элементов в начало массива и возвращает новую длину массива.

### Характеристики:

- ✅ **Добавляет в начало** — элементы добавляются в начало массива
- ✅ **Изменяет исходный массив** — мутирует массив
- ✅ **Возвращает длину** — новую длину массива
- ✅ **Множественные элементы** — можно добавить несколько элементов
- ⚠️ **Медленнее** — чем push (нужно сдвигать все элементы)

### Синтаксис:

```javascript
array.unshift(element1, element2, ..., elementN)
```

### Примеры:

```javascript
const arr = [3, 4];

arr.unshift(1, 2); // Добавить в начало
console.log(arr); // [1, 2, 3, 4]
console.log(arr.unshift(0)); // 5 (новая длина)

arr.unshift(-2, -1); // Добавить несколько элементов
console.log(arr); // [-2, -1, 0, 1, 2, 3, 4]
```

## Сравнительная таблица

| Метод | Направление | Операция | Возвращает | Производительность |
|-------|-------------|----------|------------|-------------------|
| **.push()** | Конец | Добавление | Длину | ✅ Быстро |
| **.pop()** | Конец | Удаление | Элемент | ✅ Быстро |
| **.shift()** | Начало | Удаление | Элемент | ⚠️ Медленнее |
| **.unshift()** | Начало | Добавление | Длину | ⚠️ Медленнее |

## Практические примеры

### Пример 1: Стек (LIFO - Last In First Out)

```javascript
// Стек - push и pop
const stack = [];

stack.push(1); // [1]
stack.push(2); // [1, 2]
stack.push(3); // [1, 2, 3]

const last = stack.pop(); // 3, stack = [1, 2]
const next = stack.pop(); // 2, stack = [1]
```

### Пример 2: Очередь (FIFO - First In First Out)

```javascript
// Очередь - push и shift
const queue = [];

queue.push(1); // [1]
queue.push(2); // [1, 2]
queue.push(3); // [1, 2, 3]

const first = queue.shift(); // 1, queue = [2, 3]
const next = queue.shift(); // 2, queue = [3]
```

### Пример 3: Добавление в начало

```javascript
// unshift - добавление в начало
const arr = [3, 4, 5];

arr.unshift(1, 2); // Добавить в начало
console.log(arr); // [1, 2, 3, 4, 5]
```

### Пример 4: Удаление с начала

```javascript
// shift - удаление с начала
const arr = [1, 2, 3, 4, 5];

while (arr.length > 0) {
  const item = arr.shift();
  console.log(item); // 1, 2, 3, 4, 5
}
console.log(arr); // []
```

## Производительность

### push/pop vs shift/unshift:

```javascript
// push/pop - быстро (O(1))
// Не нужно сдвигать элементы

// shift/unshift - медленнее (O(n))
// Нужно сдвигать все элементы

// Для больших массивов используйте push/pop когда возможно
```

## Лучшие практики

### ✅ Делайте:

1. **Используйте push/pop** — для стека (быстрее)
2. **Используйте push/shift** — для очереди
3. **Понимайте направление** — push/pop с конца, shift/unshift с начала
4. **Проверяйте длину** — перед pop/shift на пустом массиве

### ❌ Не делайте:

1. **Не используйте shift/unshift** — для больших массивов (медленно)
2. **Не забывайте про мутацию** — методы изменяют исходный массив
3. **Не смешивайте направления** — будьте последовательны

## Заключение

Разница между методами:

- **`.push()`** — добавляет в конец, возвращает длину
- **`.pop()`** — удаляет с конца, возвращает элемент
- **`.shift()`** — удаляет с начала, возвращает элемент
- **`.unshift()`** — добавляет в начало, возвращает длину

**Помните:** `push` и `pop` работают с концом массива и быстрее, а `shift` и `unshift` работают с началом массива и медленнее. Используйте `push/pop` для стека (LIFO), `push/shift` для очереди (FIFO), и понимайте, что все эти методы мутируют исходный массив. Для больших массивов предпочитайте `push/pop` из-за лучшей производительности.
