# Типы всплывающих окон в JavaScript?

Всплывающие окна (popup windows) в JavaScript — это окна браузера, которые открываются поверх текущего окна. Хотя современные веб-приложения редко используют всплывающие окна из-за проблем с UX и блокировщиками, понимание различных типов всплывающих окон и способов их создания критически важно для работы с браузерными API.

## Типы всплывающих окон

### 1. **alert()**

Простое модальное окно с сообщением и кнопкой OK.

```javascript
alert("Это сообщение");
// Блокирует выполнение до закрытия
```

### 2. **confirm()**

Модальное окно с сообщением и кнопками OK/Cancel.

```javascript
const result = confirm("Вы уверены?");
console.log(result); // true (OK) или false (Cancel)
```

### 3. **prompt()**

Модальное окно с полем ввода и кнопками OK/Cancel.

```javascript
const result = prompt("Введите имя:", "По умолчанию");
console.log(result); // Введенное значение или null (Cancel)
```

### 4. **window.open()**

Открывает новое окно браузера.

```javascript
const newWindow = window.open("https://example.com", "_blank");
```

## Практические примеры

### Пример 1: alert()

```javascript
// Простое уведомление
alert("Операция завершена успешно");

// С переменными
const message = "Ошибка: " + errorMessage;
alert(message);
```

### Пример 2: confirm()

```javascript
// Подтверждение действия
function deleteItem() {
  if (confirm("Вы уверены, что хотите удалить?")) {
    // Пользователь нажал OK
    performDelete();
  } else {
    // Пользователь нажал Cancel
    console.log("Удаление отменено");
  }
}
```

### Пример 3: prompt()

```javascript
// Ввод данных
function getUserName() {
  const name = prompt("Введите ваше имя:", "Гость");
  if (name) {
    console.log("Привет, " + name);
  } else {
    console.log("Имя не введено");
  }
}
```

### Пример 4: window.open()

```javascript
// Открытие нового окна
const newWindow = window.open(
  "https://example.com",
  "_blank",
  "width=800,height=600"
);

// Проверка блокировки
if (!newWindow) {
  alert("Всплывающие окна заблокированы");
}
```

## Лучшие практики

### ✅ Делайте:

1. **Используйте современные альтернативы** — модальные окна в DOM
2. **Используйте только по действию пользователя** — для window.open()
3. **Предоставляйте альтернативы** — если окна заблокированы

### ❌ Не делайте:

1. **Не используйте автоматически** — только по действию пользователя
2. **Не злоупотребляйте** — раздражает пользователей
3. **Не полагайтесь на них** — могут быть заблокированы

## Заключение

Типы всплывающих окон:

- **alert()** — простое уведомление
- **confirm()** — подтверждение
- **prompt()** — ввод данных
- **window.open()** — новое окно браузера

**Помните:** всплывающие окна устарели и имеют множество проблем. Используйте современные альтернативы (модальные окна в DOM, библиотеки) для лучшего UX. Используйте встроенные методы только для отладки или когда действительно необходимо.
