# Что такое выражения (expression) и инструкции (statement) в JavaScript?

В JavaScript существует важное различие между выражениями (expressions) и инструкциями (statements). Понимание этого различия критически важно для написания правильного кода и понимания того, как JavaScript обрабатывает код.

## Что такое выражение (Expression)?

Выражение — это код, который вычисляется в значение. Выражение всегда возвращает значение.

### Характеристики:

- ✅ **Возвращает значение** — всегда имеет результат
- ✅ **Можно использовать** — в других выражениях
- ✅ **Можно присвоить** — переменной

### Примеры:

```javascript
// Простые выражения
5; // Выражение, возвращает 5
"text"; // Выражение, возвращает "text"
true; // Выражение, возвращает true

// Арифметические выражения
5 + 3; // Выражение, возвращает 8
x * 2; // Выражение, возвращает результат

// Вызов функции (выражение)
add(5, 3); // Выражение, возвращает результат функции

// Тернарный оператор (выражение)
condition ? value1 : value2; // Выражение, возвращает значение

// Arrow function (выражение)
const add = (a, b) => a + b; // Выражение
```

## Что такое инструкция (Statement)?

Инструкция — это код, который выполняет действие, но не обязательно возвращает значение.

### Характеристики:

- ✅ **Выполняет действие** — но не возвращает значение
- ❌ **Нельзя использовать** — в других выражениях напрямую
- ✅ **Структура программы** — определяет поток выполнения

### Примеры:

```javascript
// Объявление переменной (инструкция)
let x = 5;

// if (инструкция)
if (condition) {
  // код
}

// for (инструкция)
for (let i = 0; i < 10; i++) {
  // код
}

// function declaration (инструкция)
function test() {
  // код
}

// return (инструкция)
return value;
```

## Ключевые различия

### 1. **Возврат значения**

```javascript
// Выражение - возвращает значение
5 + 3; // 8

// Инструкция - не возвращает значение
let x = 5; // undefined (не возвращает значение)
```

### 2. **Использование в других конструкциях**

```javascript
// Выражение - можно использовать
const result = 5 + 3; // OK
const value = condition ? a : b; // OK

// Инструкция - нельзя использовать напрямую
const x = if (condition) { a } else { b }; // SyntaxError
```

### 3. **Function Expression vs Function Statement**

```javascript
// Function Expression (выражение)
const func = function() {
  return "test";
};

// Function Statement (инструкция)
function test() {
  return "test";
}
```

## Практические примеры

### Пример 1: Выражения

```javascript
// Все это выражения
const a = 5; // 5 - выражение
const b = a + 3; // a + 3 - выражение
const c = Math.max(a, b); // Math.max(a, b) - выражение
const d = condition ? a : b; // Тернарный оператор - выражение
```

### Пример 2: Инструкции

```javascript
// Все это инструкции
let x = 5; // Объявление переменной - инструкция
if (x > 0) { // if - инструкция
  console.log("Positive");
}
for (let i = 0; i < 10; i++) { // for - инструкция
  console.log(i);
}
```

### Пример 3: Смешанные случаи

```javascript
// Инструкция может содержать выражения
let result = 5 + 3; // let - инструкция, 5 + 3 - выражение

if (x > 0) { // if - инструкция, x > 0 - выражение
  console.log("Positive");
}
```

## Expression Statements

Некоторые выражения могут быть использованы как инструкции:

```javascript
// Выражение как инструкция
5 + 3; // Выражение, но используется как инструкция (не имеет эффекта)
console.log("test"); // Выражение (вызов функции), используется как инструкция
```

## Лучшие практики

### ✅ Делайте:

1. **Понимайте разницу** — между выражениями и инструкциями
2. **Используйте выражения** — когда нужно значение
3. **Используйте инструкции** — для управления потоком

## Заключение

Выражения и инструкции:

- **Выражение** — возвращает значение, можно использовать в других выражениях
- **Инструкция** — выполняет действие, определяет структуру программы
- **Разные цели** — выражения для значений, инструкции для действий
- **Могут комбинироваться** — инструкции могут содержать выражения

**Помните:** понимание разницы между выражениями и инструкциями критически важно для написания правильного JavaScript кода. Выражения возвращают значения и могут использоваться в других выражениях, а инструкции выполняют действия и определяют структуру программы.
