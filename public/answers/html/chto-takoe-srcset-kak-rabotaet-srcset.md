# Что такое `srcset`? Как работает `srcset`?

`srcset` — это атрибут для элементов `<img>` и `<source>`, который позволяет указать несколько версий изображения для разных условий (размер экрана, плотность пикселей). Браузер выбирает наиболее подходящее изображение.

## Что такое srcset?

`srcset` — это список источников изображений с описанием условий, при которых каждое изображение должно использоваться.

## Синтаксис

### С дескрипторами ширины:

```html
<img srcset="small.jpg 480w,
             medium.jpg 768w,
             large.jpg 1200w"
     src="small.jpg"
     alt="Изображение">
```

### С дескрипторами плотности:

```html
<img srcset="image-1x.jpg 1x,
             image-2x.jpg 2x,
             image-3x.jpg 3x"
     src="image-1x.jpg"
     alt="Изображение">
```

## Как работает srcset

### 1. **Браузер анализирует условия**

- Размер viewport
- Плотность пикселей экрана
- Доступная пропускная способность

### 2. **Выбирает подходящее изображение**

Браузер выбирает изображение, которое лучше всего подходит для текущих условий.

### 3. **Загружает выбранное изображение**

Только одно изображение загружается, что улучшает производительность.

## Примеры использования

### Адаптивные изображения:

```html
<img srcset="photo-small.jpg 480w,
             photo-medium.jpg 768w,
             photo-large.jpg 1200w"
     sizes="(max-width: 600px) 480px,
            (max-width: 900px) 768px,
            1200px"
     src="photo-small.jpg"
     alt="Фото">
```

### Retina дисплеи:

```html
<img srcset="logo-1x.png 1x,
             logo-2x.png 2x"
     src="logo-1x.png"
     alt="Логотип">
```

### Комбинация с `<picture>`:

```html
<picture>
    <source srcset="image-large.webp" type="image/webp" media="(min-width: 1200px)">
    <source srcset="image-medium.webp" type="image/webp" media="(min-width: 768px)">
    <img srcset="image-small.jpg 480w,
                 image-medium.jpg 768w,
                 image-large.jpg 1200w"
         src="image-small.jpg"
         alt="Изображение">
</picture>
```

## Атрибут sizes

`sizes` указывает размер изображения в разных условиях:

```html
<img srcset="photo-480.jpg 480w,
             photo-768.jpg 768w,
             photo-1200.jpg 1200w"
     sizes="(max-width: 600px) 100vw,
            (max-width: 900px) 50vw,
            33vw"
     src="photo-480.jpg"
     alt="Фото">
```

## Заключение

**`srcset` используется для:**

1. ✅ Адаптивных изображений
2. ✅ Оптимизации производительности
3. ✅ Поддержки Retina дисплеев
4. ✅ Разных изображений для разных устройств

**Как работает:**
- Браузер анализирует условия
- Выбирает подходящее изображение
- Загружает только выбранное

**Преимущества:**
- Улучшенная производительность
- Адаптивность
- Лучший UX на разных устройствах

`srcset` — современный способ оптимизации изображений для разных устройств.
