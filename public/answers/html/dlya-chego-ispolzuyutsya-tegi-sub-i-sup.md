# Для чего используются тэги `<sub>` и `<sup>`?

`<sub>` и `<sup>` — это HTML элементы для создания нижнего и верхнего индекса соответственно. Они используются для математических формул, химических формул, сносок и других случаев, где нужны индексы.

## Назначение тегов

### **`<sub>`** — Нижний индекс (Subscript)

Используется для текста, который должен отображаться ниже базовой линии.

```html
H<sub>2</sub>O
```

**Результат:** H₂O

### **`<sup>`** — Верхний индекс (Superscript)

Используется для текста, который должен отображаться выше базовой линии.

```html
E = mc<sup>2</sup>
```

**Результат:** E = mc²

## Примеры использования

### Химические формулы:

```html
<p>Вода: H<sub>2</sub>O</p>
<p>Углекислый газ: CO<sub>2</sub></p>
<p>Серная кислота: H<sub>2</sub>SO<sub>4</sub></p>
```

### Математические формулы:

```html
<p>Квадрат: x<sup>2</sup></p>
<p>Степень: 2<sup>3</sup> = 8</p>
<p>Формула: E = mc<sup>2</sup></p>
```

### Сноски:

```html
<p>Текст со сноской<sup>1</sup></p>
<p><sup>1</sup> Примечание</p>
```

### Даты:

```html
<p>1<sup>st</sup> января</p>
<p>2<sup>nd</sup> февраля</p>
<p>3<sup>rd</sup> марта</p>
```

## Стилизация

### Базовые стили:

```css
sub {
    font-size: 0.75em;
    vertical-align: sub;
}

sup {
    font-size: 0.75em;
    vertical-align: super;
}
```

### Кастомная стилизация:

```css
sub {
    font-size: 0.8em;
    color: #666;
}

sup {
    font-size: 0.8em;
    color: blue;
    font-weight: bold;
}
```

## Семантика

`<sub>` и `<sup>` имеют семантическое значение:
- Улучшают доступность
- Помогают скринридерам правильно читать формулы
- Улучшают SEO

## Альтернативы

### Через CSS (не рекомендуется):

```css
.lower {
    font-size: 0.75em;
    vertical-align: sub;
}
```

**Проблема:** Нет семантического значения.

## Заключение

**`<sub>` и `<sup>` используются для:**

1. ✅ Химических формул (H₂O, CO₂)
2. ✅ Математических формул (x², E=mc²)
3. ✅ Сносок и примечаний
4. ✅ Дат (1st, 2nd, 3rd)

**Преимущества:**
- Семантическая разметка
- Улучшенная доступность
- Правильное отображение
- Поддержка скринридеров

**Рекомендации:**
- Используйте для формул и сносок
- Не используйте только для визуального эффекта
- Комбинируйте с CSS для стилизации

`<sub>` и `<sup>` — семантически правильные элементы для индексов в тексте.
