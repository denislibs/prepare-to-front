# Что такое и зачем нужен атрибут `rel="nofollow"` в HTML?

Атрибут `rel="nofollow"` в HTML используется для указания поисковым системам не переходить по ссылке и не передавать "вес" страницы на связанный ресурс.

## Что такое rel="nofollow"?

`rel="nofollow"` — это значение атрибута `rel` для элементов `<a>` и `<area>`, которое сообщает поисковым системам не индексировать ссылку.

## Использование

### Базовый синтаксис:

```html
<a href="https://example.com" rel="nofollow">Ссылка</a>
```

## Зачем нужен?

### 1. **Защита от спама**

```html
<!-- Комментарии пользователей -->
<a href="https://spam-site.com" rel="nofollow">Ссылка</a>
```

### 2. **Платные ссылки**

```html
<!-- Рекламные ссылки -->
<a href="https://advertiser.com" rel="nofollow">Реклама</a>
```

### 3. **Ненадежный контент**

```html
<!-- Ссылки на непроверенные источники -->
<a href="https://unverified.com" rel="nofollow">Источник</a>
```

## Примеры

### Комментарии:

```html
<div class="comment">
    <p>Отличная статья!</p>
    <a href="https://user-site.com" rel="nofollow">Мой сайт</a>
</div>
```

### Платные ссылки:

```html
<article>
    <p>Текст статьи...</p>
    <a href="https://sponsor.com" rel="nofollow sponsored">Спонсор</a>
</article>
```

### UGC (User Generated Content):

```html
<div class="user-content">
    <a href="https://user-link.com" rel="nofollow ugc">Пользовательская ссылка</a>
</div>
```

## Комбинации значений

### `nofollow` + `sponsored`:

```html
<a href="https://ad.com" rel="nofollow sponsored">Реклама</a>
```

### `nofollow` + `ugc`:

```html
<a href="https://user.com" rel="nofollow ugc">Пользовательский контент</a>
```

## Влияние на SEO

### Что происходит:

1. ✅ Поисковики не переходят по ссылке
2. ✅ Не передается "вес" страницы
3. ✅ Ссылка не индексируется

### Что не происходит:

- ❌ Ссылка не скрывается от пользователей
- ❌ Ссылка работает нормально
- ❌ Не влияет на кликабельность

## Заключение

**`rel="nofollow"` используется для:**

1. ✅ Защиты от спама в комментариях
2. ✅ Маркировки платных ссылок
3. ✅ Указания на ненадежный контент

**Значения:**
- `nofollow` — не переходить по ссылке
- `sponsored` — платная ссылка
- `ugc` — пользовательский контент

**Рекомендации:**
- Используйте для пользовательского контента
- Используйте для платных ссылок
- Комбинируйте с `sponsored` и `ugc`

Правильное использование `rel="nofollow"` помогает защитить SEO и маркировать контент.
