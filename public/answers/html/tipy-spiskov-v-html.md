# Типы списков в HTML?

В HTML существует три основных типа списков: упорядоченные (`<ol>`), неупорядоченные (`<ul>`) и списки определений (`<dl>`). Каждый тип списка имеет свое назначение и семантическое значение.

## Типы списков

### 1. **Неупорядоченный список (`<ul>`)**

Используется для списков, где порядок элементов не важен. Элементы отображаются с маркерами (bullets).

**Структура:**
```html
<ul>
    <li>Элемент списка 1</li>
    <li>Элемент списка 2</li>
    <li>Элемент списка 3</li>
</ul>
```

**Примеры:**
```html
<!-- Список покупок -->
<ul>
    <li>Хлеб</li>
    <li>Молоко</li>
    <li>Яйца</li>
</ul>

<!-- Навигационное меню -->
<nav>
    <ul>
        <li><a href="/">Главная</a></li>
        <li><a href="/about">О нас</a></li>
        <li><a href="/contact">Контакты</a></li>
    </ul>
</nav>
```

**Стили маркеров:**
```css
ul { list-style-type: disc; }    /* Круг (по умолчанию) */
ul { list-style-type: circle; }  /* Круглый */
ul { list-style-type: square; }  /* Квадрат */
ul { list-style-type: none; }   /* Без маркера */
```

### 2. **Упорядоченный список (`<ol>`)**

Используется для списков, где порядок элементов важен. Элементы отображаются с номерами или буквами.

**Структура:**
```html
<ol>
    <li>Первый элемент</li>
    <li>Второй элемент</li>
    <li>Третий элемент</li>
</ol>
```

**Примеры:**
```html
<!-- Инструкции -->
<ol>
    <li>Открыть файл</li>
    <li>Прочитать содержимое</li>
    <li>Сохранить изменения</li>
</ol>

<!-- Рейтинг -->
<ol>
    <li>Первое место</li>
    <li>Второе место</li>
    <li>Третье место</li>
</ol>
```

**Типы нумерации:**
```html
<!-- Числа (по умолчанию) -->
<ol type="1">
    <li>Элемент</li>
</ol>

<!-- Заглавные буквы -->
<ol type="A">
    <li>Элемент</li>
</ol>

<!-- Строчные буквы -->
<ol type="a">
    <li>Элемент</li>
</ol>

<!-- Римские цифры (заглавные) -->
<ol type="I">
    <li>Элемент</li>
</ol>

<!-- Римские цифры (строчные) -->
<ol type="i">
    <li>Элемент</li>
</ol>
```

**Начало нумерации:**
```html
<ol start="5">
    <li>Пятый элемент</li>
    <li>Шестой элемент</li>
</ol>
```

**Обратная нумерация:**
```html
<ol reversed>
    <li>Третий элемент</li>
    <li>Второй элемент</li>
    <li>Первый элемент</li>
</ol>
```

**CSS стили:**
```css
ol { list-style-type: decimal; }        /* 1, 2, 3 */
ol { list-style-type: decimal-leading-zero; } /* 01, 02, 03 */
ol { list-style-type: lower-roman; }   /* i, ii, iii */
ol { list-style-type: upper-roman; }  /* I, II, III */
ol { list-style-type: lower-alpha; }   /* a, b, c */
ol { list-style-type: upper-alpha; }  /* A, B, C */
```

### 3. **Список определений (`<dl>`)**

Используется для списков терминов и их определений (словари, глоссарии).

**Структура:**
```html
<dl>
    <dt>Термин</dt>
    <dd>Определение термина</dd>
    
    <dt>Другой термин</dt>
    <dd>Определение другого термина</dd>
</dl>
```

**Примеры:**
```html
<!-- Глоссарий -->
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language — язык разметки веб-страниц</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets — язык стилизации веб-страниц</dd>
    
    <dt>JavaScript</dt>
    <dd>Язык программирования для создания интерактивных веб-страниц</dd>
</dl>

<!-- Метаданные -->
<dl>
    <dt>Автор</dt>
    <dd>Иван Иванов</dd>
    
    <dt>Дата публикации</dt>
    <dd>15 января 2024</dd>
    
    <dt>Категория</dt>
    <dd>Веб-разработка</dd>
</dl>
```

**Множественные определения:**
```html
<dl>
    <dt>Браузер</dt>
    <dd>Google Chrome</dd>
    <dd>Mozilla Firefox</dd>
    <dd>Microsoft Edge</dd>
</dl>
```

**Множественные термины:**
```html
<dl>
    <dt>HTML</dt>
    <dt>HyperText Markup Language</dt>
    <dd>Язык разметки веб-страниц</dd>
</dl>
```

## Вложенные списки

### Вложенные неупорядоченные списки:

```html
<ul>
    <li>Пункт 1
        <ul>
            <li>Подпункт 1.1</li>
            <li>Подпункт 1.2</li>
        </ul>
    </li>
    <li>Пункт 2</li>
</ul>
```

### Вложенные упорядоченные списки:

```html
<ol>
    <li>Шаг 1
        <ol>
            <li>Подшаг 1.1</li>
            <li>Подшаг 1.2</li>
        </ol>
    </li>
    <li>Шаг 2</li>
</ol>
```

### Смешанные списки:

```html
<ol>
    <li>Глава 1
        <ul>
            <li>Раздел 1.1</li>
            <li>Раздел 1.2</li>
        </ul>
    </li>
    <li>Глава 2</li>
</ol>
```

## Стилизация списков

### Базовые стили:

```css
ul, ol {
    margin: 1em 0;
    padding-left: 40px;
}

li {
    margin: 0.5em 0;
}

dl {
    margin: 1em 0;
}

dt {
    font-weight: bold;
    margin-top: 1em;
}

dd {
    margin-left: 2em;
    margin-bottom: 0.5em;
}
```

### Кастомные маркеры:

```css
ul.custom {
    list-style-type: none;
    padding-left: 0;
}

ul.custom li::before {
    content: "✓ ";
    color: green;
    font-weight: bold;
    margin-right: 0.5em;
}
```

### Горизонтальные списки:

```css
ul.horizontal {
    list-style-type: none;
    padding: 0;
    display: flex;
    gap: 1em;
}

ul.horizontal li {
    margin: 0;
}
```

## Доступность

### Правильная доступность:

```html
<nav aria-label="Основная навигация">
    <ul>
        <li><a href="/">Главная</a></li>
        <li><a href="/about">О нас</a></li>
    </ul>
</nav>
```

### Списки определений для доступности:

```html
<dl>
    <dt>Email</dt>
    <dd><a href="mailto:info@example.com">info@example.com</a></dd>
    
    <dt>Телефон</dt>
    <dd><a href="tel:+1234567890">+1 (234) 567-890</a></dd>
</dl>
```

## Практические примеры

### Навигационное меню:

```html
<nav>
    <ul class="nav-menu">
        <li><a href="/">Главная</a></li>
        <li><a href="/products">Продукты</a></li>
        <li><a href="/about">О нас</a></li>
        <li><a href="/contact">Контакты</a></li>
    </ul>
</nav>
```

### Хлебные крошки:

```html
<nav aria-label="Хлебные крошки">
    <ol class="breadcrumb">
        <li><a href="/">Главная</a></li>
        <li><a href="/category">Категория</a></li>
        <li aria-current="page">Текущая страница</li>
    </ol>
</nav>
```

### Инструкции:

```html
<article>
    <h2>Как приготовить кофе</h2>
    <ol>
        <li>Вскипятите воду</li>
        <li>Добавьте кофе в фильтр</li>
        <li>Залейте горячей водой</li>
        <li>Дождитесь готовности</li>
    </ol>
</article>
```

### Глоссарий:

```html
<section>
    <h2>Глоссарий терминов</h2>
    <dl>
        <dt>API</dt>
        <dd>Application Programming Interface — интерфейс программирования приложений</dd>
        
        <dt>REST</dt>
        <dd>Representational State Transfer — архитектурный стиль веб-сервисов</dd>
    </dl>
</section>
```

## Заключение

**Типы списков в HTML:**

1. **`<ul>`** — неупорядоченный список
   - Для списков без важного порядка
   - Маркеры (bullets)
   - Используется для навигации, перечней

2. **`<ol>`** — упорядоченный список
   - Для списков с важным порядком
   - Нумерация (числа, буквы, римские цифры)
   - Используется для инструкций, рейтингов

3. **`<dl>`** — список определений
   - Для терминов и их определений
   - Состоит из `<dt>` (термин) и `<dd>` (определение)
   - Используется для глоссариев, метаданных

**Ключевые моменты:**
- ✅ Используйте семантически правильный тип списка
- ✅ Списки можно вкладывать друг в друга
- ✅ Стилизуйте через CSS для нужного внешнего вида
- ✅ Обеспечивайте доступность через ARIA атрибуты

Правильный выбор типа списка улучшает семантику, доступность и SEO вашего HTML-кода.
