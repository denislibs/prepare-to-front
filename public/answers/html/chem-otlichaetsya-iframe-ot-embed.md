# Чем отличается `<iframe>` от `<embed>`?

`<iframe>` и `<embed>` — это HTML-элементы для встраивания контента, но они имеют разные назначения, возможности и использование. Понимание различий помогает выбрать правильный элемент.

## Элемент `<iframe>`

### Что такое `<iframe>`?

`<iframe>` (Inline Frame) — это элемент для встраивания другой HTML-страницы в текущую страницу.

### Использование:

```html
<iframe src="https://example.com" width="600" height="400"></iframe>
```

### Особенности:

- Встраивает HTML-страницы
- Имеет собственный контекст документа
- Поддерживает атрибуты безопасности (`sandbox`)
- Широко поддерживается браузерами

## Элемент `<embed>`

### Что такое `<embed>`?

`<embed>` — это элемент для встраивания внешнего контента, такого как плагины, мультимедиа или приложения.

### Использование:

```html
<embed src="video.swf" type="application/x-shockwave-flash" width="600" height="400">
```

### Особенности:

- Встраивает внешний контент (плагины, мультимедиа)
- Не имеет закрывающего тега
- Менее используемый в современной веб-разработке
- Заменяется более современными элементами

## Ключевые различия

### 1. **Назначение**

**`<iframe>`:**
- Встраивание HTML-страниц
- Встраивание веб-контента

**`<embed>`:**
- Встраивание плагинов
- Встраивание мультимедиа (устарело)

### 2. **Синтаксис**

**`<iframe>`:**
```html
<iframe src="page.html"></iframe>
```

**`<embed>`:**
```html
<embed src="content.swf" type="application/x-shockwave-flash">
<!-- Самозакрывающийся тег -->
```

### 3. **Поддержка**

**`<iframe>`:**
- ✅ Широко поддерживается
- ✅ Активно используется

**`<embed>`:**
- ⚠️ Менее поддерживается
- ⚠️ Устаревает

### 4. **Безопасность**

**`<iframe>`:**
- Поддерживает `sandbox`
- Поддерживает `allow`
- Лучшая изоляция

**`<embed>`:**
- Ограниченная безопасность
- Меньше контроля

## Примеры использования

### `<iframe>` — Встраивание страницы:

```html
<iframe 
    src="https://example.com" 
    width="600" 
    height="400"
    frameborder="0"
    allowfullscreen>
</iframe>
```

### `<iframe>` — YouTube видео:

```html
<iframe 
    src="https://www.youtube.com/embed/VIDEO_ID" 
    width="560" 
    height="315"
    frameborder="0"
    allowfullscreen>
</iframe>
```

### `<embed>` — Flash (устарело):

```html
<embed 
    src="animation.swf" 
    type="application/x-shockwave-flash"
    width="600"
    height="400">
```

## Современные альтернативы

### Для мультимедиа:

```html
<!-- Видео -->
<video src="video.mp4" controls></video>

<!-- Аудио -->
<audio src="audio.mp3" controls></audio>
```

### Для встраивания:

```html
<!-- Iframe для веб-контента -->
<iframe src="https://example.com"></iframe>

<!-- Object для других типов -->
<object data="document.pdf" type="application/pdf"></object>
```

## Когда использовать

### Используйте `<iframe>` для:

- Встраивания веб-страниц
- Встраивания виджетов
- Встраивания карт
- Встраивания видео (YouTube, Vimeo)

### Используйте `<embed>` для:

- ⚠️ Устаревших плагинов (не рекомендуется)
- ⚠️ Flash-контента (устарело)

## Заключение

**`<iframe>`:**

1. ✅ Встраивание HTML-страниц
2. ✅ Широкая поддержка
3. ✅ Лучшая безопасность
4. ✅ Активное использование

**`<embed>`:**

1. ⚠️ Встраивание плагинов (устарело)
2. ⚠️ Менее поддерживается
3. ⚠️ Ограниченная безопасность
4. ⚠️ Заменяется современными элементами

**Различия:**

- Назначение: HTML-страницы vs плагины
- Поддержка: Широкая vs ограниченная
- Безопасность: Лучшая vs ограниченная
- Использование: Активное vs устаревающее

**Рекомендации:**

- Используйте `<iframe>` для веб-контента
- Избегайте `<embed>` для нового контента
- Используйте современные элементы (`<video>`, `<audio>`) для мультимедиа

В современной веб-разработке `<iframe>` предпочтительнее `<embed>` для большинства задач.
