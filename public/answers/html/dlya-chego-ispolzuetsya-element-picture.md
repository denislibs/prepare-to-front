# Для чего используется элемент `<picture>`?

`<picture>` — это HTML5 элемент, который позволяет предоставлять несколько версий изображения для разных условий (размер экрана, плотность пикселей, формат). Это улучшает производительность и адаптивность.

## Что такое `<picture>`?

`<picture>` — это контейнер для нескольких источников изображений, который позволяет браузеру выбрать наиболее подходящее изображение на основе условий.

## Базовое использование

```html
<picture>
    <source srcset="large.jpg" media="(min-width: 1200px)">
    <source srcset="medium.jpg" media="(min-width: 768px)">
    <img src="small.jpg" alt="Описание">
</picture>
```

## Примеры использования

### Адаптивные изображения:

```html
<picture>
    <source srcset="photo-large.jpg" media="(min-width: 1200px)">
    <source srcset="photo-medium.jpg" media="(min-width: 768px)">
    <img src="photo-small.jpg" alt="Фото">
</picture>
```

### Разные форматы:

```html
<picture>
    <source srcset="image.avif" type="image/avif">
    <source srcset="image.webp" type="image/webp">
    <img src="image.jpg" alt="Изображение">
</picture>
```

### Плотность пикселей:

```html
<picture>
    <source srcset="image-2x.jpg" media="(min-resolution: 2dppx)">
    <img src="image-1x.jpg" alt="Изображение">
</picture>
```

## Преимущества

### 1. **Производительность**

Загрузка только нужного изображения для устройства.

### 2. **Адаптивность**

Разные изображения для разных размеров экрана.

### 3. **Современные форматы**

Поддержка AVIF, WebP с fallback на JPEG/PNG.
