# Разница между `<meter>` и `<progress>`?

`<meter>` и `<progress>` — это HTML5 элементы для отображения значений, но они имеют разные назначения: `<meter>` для статических измерений в известном диапазоне, а `<progress>` для динамического прогресса выполнения задачи.

## Разница между `<meter>` и `<progress>`

### **`<meter>`** — Измеритель

Используется для отображения скалярного значения в известном диапазоне (например, использование диска, рейтинг).

```html
<meter value="0.6" min="0" max="1">60%</meter>
<meter value="75" min="0" max="100">75%</meter>
```

**Характеристики:**
- Статическое значение
- Известный диапазон
- Для измерений и рейтингов

### **`<progress>`** — Прогресс

Используется для отображения прогресса выполнения задачи (например, загрузка файла, обработка данных).

```html
<progress value="70" max="100">70%</progress>
<progress>Загрузка...</progress>
```

**Характеристики:**
- Динамическое значение
- Прогресс выполнения
- Для задач и процессов

## Сравнение

| Характеристика | `<meter>` | `<progress>` |
|----------------|-----------|--------------|
| **Назначение** | Измерения | Прогресс задачи |
| **Значение** | Статическое | Динамическое |
| **Использование** | Рейтинги, использование | Загрузка, обработка |

## Примеры использования

### `<meter>` — Рейтинг:

```html
<label>Рейтинг: 
    <meter value="4" min="0" max="5">4 из 5</meter>
</label>
```

### `<meter>` — Использование диска:

```html
<label>Использование диска: 
    <meter value="60" min="0" max="100">60%</meter>
</label>
```

### `<progress>` — Загрузка:

```html
<label>Загрузка файла: 
    <progress value="70" max="100">70%</progress>
</label>
```

### `<progress>` — Неопределенный прогресс:

```html
<progress>Обработка...</progress>
```

## Атрибуты

### `<meter>`:

```html
<meter value="75" min="0" max="100" low="25" high="75" optimum="50">
    75%
</meter>
```

- `value` — текущее значение
- `min` — минимальное значение
- `max` — максимальное значение
- `low` — низкое значение
- `high` — высокое значение
- `optimum` — оптимальное значение

### `<progress>`:

```html
<progress value="50" max="100">50%</progress>
```

- `value` — текущее значение
- `max` — максимальное значение (по умолчанию 1)

## Заключение

**Разница:**

- **`<meter>`**: Статические измерения в известном диапазоне (рейтинги, использование)
- **`<progress>`**: Динамический прогресс выполнения задачи (загрузка, обработка)

**Использование:**
- `<meter>` для рейтингов, использования ресурсов
- `<progress>` для загрузки, обработки данных

Правильный выбор элемента улучшает семантику и доступность.
