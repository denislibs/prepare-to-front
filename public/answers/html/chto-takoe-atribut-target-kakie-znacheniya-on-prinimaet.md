# Что такое атрибут `target`? Какие значения он принимает?

Атрибут `target` в HTML определяет, где открыть ссылку или результат формы. Он используется в элементах `<a>` и `<form>` для управления поведением открытия ссылок.

## Что такое target?

`target` — это атрибут, который указывает браузеру, где открыть документ при переходе по ссылке или отправке формы.

## Значения атрибута

### 1. **`_self`** — Текущее окно (по умолчанию)

```html
<a href="page.html" target="_self">Ссылка</a>
```

- Открывает в том же окне/вкладке
- Значение по умолчанию

### 2. **`_blank`** — Новое окно/вкладка

```html
<a href="page.html" target="_blank">Ссылка</a>
```

- Открывает в новой вкладке
- Рекомендуется использовать с `rel="noopener noreferrer"`

### 3. **`_parent`** — Родительский фрейм

```html
<a href="page.html" target="_parent">Ссылка</a>
```

- Открывает в родительском фрейме
- Используется в контексте фреймов

### 4. **`_top`** — Верхний уровень

```html
<a href="page.html" target="_top">Ссылка</a>
```

- Открывает в верхнем уровне (выходит из всех фреймов)
- Используется в контексте фреймов

### 5. **Имя фрейма/окна**

```html
<a href="page.html" target="myFrame">Ссылка</a>
```

- Открывает в фрейме с указанным именем
- Или в окне с указанным именем

## Примеры использования

### Внешние ссылки:

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
    Внешний сайт
</a>
```

### Внутренние ссылки:

```html
<a href="/page.html" target="_self">Внутренняя ссылка</a>
<!-- или просто -->
<a href="/page.html">Внутренняя ссылка</a>
```

### Формы:

```html
<form action="/submit" method="POST" target="_blank">
    <!-- Форма откроется в новой вкладке -->
</form>
```

## Безопасность

### С `target="_blank"`:

```html
<!-- Безопасно -->
<a href="https://external.com" target="_blank" rel="noopener noreferrer">
    Ссылка
</a>
```

**Почему `rel="noopener noreferrer"`:**
- `noopener` — предотвращает доступ нового окна к `window.opener`
- `noreferrer` — не передает referrer информацию

## Заключение

**Значения `target`:**

1. **`_self`** — текущее окно (по умолчанию)
2. **`_blank`** — новое окно/вкладка
3. **`_parent`** — родительский фрейм
4. **`_top`** — верхний уровень
5. **Имя фрейма** — конкретный фрейм/окно

**Рекомендации:**
- Используйте `_self` для внутренних ссылок
- Используйте `_blank` с `rel="noopener noreferrer"` для внешних ссылок
- Избегайте использования в современных приложениях без необходимости

Правильное использование `target` улучшает UX и безопасность.
