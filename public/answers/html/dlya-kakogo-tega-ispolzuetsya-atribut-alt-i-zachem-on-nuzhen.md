# Для какого тэга используется атрибут `alt` и зачем он нужен?

Атрибут `alt` используется для элемента `<img>` и является обязательным атрибутом для обеспечения доступности и правильной работы веб-страниц. Он предоставляет альтернативный текст для изображений, который отображается, если изображение не загрузилось, или читается скринридерами для людей с нарушениями зрения.

## Для какого тега используется `alt`?

Атрибут `alt` используется для элемента **`<img>`** (изображение):

```html
<img src="photo.jpg" alt="Описание изображения">
```

## Зачем нужен атрибут `alt`?

### 1. **Доступность (Accessibility)**

Главная цель `alt` — обеспечить доступность контента для людей с нарушениями зрения, использующих скринридеры (screen readers):

```html
<img src="chart.png" alt="График показывает рост продаж на 25% за последний квартал">
```

Скринридер прочитает: "График показывает рост продаж на 25% за последний квартал"

### 2. **Резервный текст**

Если изображение не загрузилось (медленное соединение, ошибка загрузки, неверный путь), браузер отобразит текст из `alt`:

```html
<img src="missing-image.jpg" alt="Фотография заката над океаном">
```

**Результат:** Вместо сломанной иконки отобразится текст "Фотография заката над океаном"

### 3. **SEO (Поисковая оптимизация)**

Поисковые системы используют `alt` для понимания содержания изображения и улучшения индексации:

```html
<img src="product.jpg" alt="Ноутбук MacBook Pro 16 дюймов">
```

Это помогает изображению появляться в результатах поиска по изображениям.

### 4. **Контекст для пользователей**

`alt` помогает пользователям понять назначение изображения, даже если оно еще не загрузилось:

```html
<img src="loading.gif" alt="Загрузка данных...">
```

## Правильное использование `alt`

### ✅ Хорошие примеры:

```html
<!-- Описательный alt для информативного изображения -->
<img src="team-photo.jpg" alt="Команда разработчиков из 5 человек на фоне офиса">

<!-- Функциональное изображение (кнопка) -->
<img src="search-icon.svg" alt="Поиск">

<!-- Декоративное изображение -->
<img src="decoration.png" alt="">

<!-- График или диаграмма -->
<img src="sales-chart.png" alt="График продаж: январь - 10000, февраль - 15000, март - 20000">
```

### ❌ Плохие примеры:

```html
<!-- Слишком общий -->
<img src="photo.jpg" alt="Изображение">

<!-- Избыточный текст -->
<img src="logo.png" alt="Логотип компании Example Corp, основанной в 2020 году, специализирующейся на...">

<!-- Плохо: alt повторяет окружающий текст -->
<p>Фотография заката</p>
<img src="sunset.jpg" alt="Фотография заката">

<!-- Плохо: отсутствует alt -->
<img src="photo.jpg">
```

## Типы изображений и `alt`

### 1. **Информативные изображения**

Содержат важную информацию:

```html
<img src="diagram.png" alt="Схема работы: входные данные → обработка → результат">
```

**Правило:** `alt` должен описывать содержание и назначение изображения.

### 2. **Декоративные изображения**

Используются только для оформления:

```html
<img src="divider.png" alt="">
```

**Правило:** Используйте пустой `alt=""`, чтобы скринридер пропустил изображение.

### 3. **Функциональные изображения**

Используются как кнопки или ссылки:

```html
<a href="/search">
    <img src="search-icon.svg" alt="Поиск">
</a>

<button>
    <img src="close-icon.svg" alt="Закрыть">
</button>
```

**Правило:** `alt` должен описывать функцию, а не внешний вид.

### 4. **Изображения с текстом**

Содержат текст, который нужно передать:

```html
<img src="banner.png" alt="Скидка 50% на все товары до конца месяца">
```

**Правило:** `alt` должен содержать весь текст из изображения.

### 5. **Сложные изображения (графики, диаграммы)**

Требуют детального описания:

```html
<img src="chart.png" alt="Круговая диаграмма: Продукт A - 40%, Продукт B - 35%, Продукт C - 25%">
```

**Правило:** `alt` должен передавать ключевую информацию из графика.

## Длина `alt` текста

### Рекомендации:

- **Короткий:** 1-5 слов для простых изображений
- **Средний:** До 125 символов для описательных изображений
- **Длинный:** Для сложных изображений можно использовать `<figure>` с `<figcaption>`

```html
<!-- Короткий -->
<img src="icon.png" alt="Иконка">

<!-- Средний -->
<img src="photo.jpg" alt="Группа людей на конференции по веб-разработке">

<!-- Длинный - лучше использовать figure -->
<figure>
    <img src="complex-chart.png" alt="График продаж по месяцам">
    <figcaption>
        Детальный график показывает рост продаж: январь - 10000, 
        февраль - 15000, март - 20000, апрель - 25000
    </figcaption>
</figure>
```

## Специальные случаи

### Изображения в ссылках:

```html
<a href="/product">
    <img src="product.jpg" alt="Ноутбук MacBook Pro - перейти к описанию">
</a>
```

### Изображения с `title`:

```html
<img src="photo.jpg" 
     alt="Основное описание для скринридеров"
     title="Дополнительная информация при наведении">
```

**Важно:** `alt` обязателен, `title` опционален. `alt` используется для доступности, `title` для дополнительной информации.

### Изображения с `srcset`:

```html
<img src="photo-small.jpg"
     srcset="photo-small.jpg 480w, photo-large.jpg 1200w"
     alt="Описание изображения">
```

`alt` применяется ко всем размерам изображения.

## Проверка `alt` атрибутов

### Инструменты:

1. **WAVE** (Web Accessibility Evaluation Tool)
2. **axe DevTools**
3. **Lighthouse** (Chrome DevTools)
4. **Валидатор W3C**

### Ручная проверка:

1. Отключите изображения в браузере
2. Проверьте, что контент понятен
3. Используйте скринридер для проверки

## Заключение

**Атрибут `alt` используется для:**
- Элемента `<img>` (изображения)

**Зачем нужен:**
- ✅ **Доступность** — для скринридеров
- ✅ **Резервный текст** — если изображение не загрузилось
- ✅ **SEO** — для поисковых систем
- ✅ **Контекст** — помощь пользователям

**Правила использования:**
- Всегда добавляйте `alt` к `<img>`
- Используйте описательный текст для информативных изображений
- Используйте пустой `alt=""` для декоративных изображений
- Описывайте функцию для функциональных изображений
- Держите текст кратким, но информативным

**Обязательность:**
- `alt` является **обязательным** атрибутом для `<img>` в HTML5
- Отсутствие `alt` делает HTML невалидным
- Это критично для доступности веб-контента

Правильное использование `alt` — это не только требование стандартов, но и важная практика для создания доступных и качественных веб-сайтов.
