# Для чего используется атрибут `pattern`?

Атрибут `pattern` в HTML определяет регулярное выражение, которому должно соответствовать значение поля формы. Он используется для валидации ввода на стороне клиента.

## Что такое pattern?

`pattern` — это атрибут для элементов `<input>`, который задает регулярное выражение для валидации значения.

## Использование

### Базовый синтаксис:

```html
<input type="text" pattern="[0-9]{4}" title="4 цифры">
```

## Примеры

### Телефон:

```html
<input type="tel" 
       pattern="[0-9]{10}" 
       title="10 цифр"
       placeholder="1234567890">
```

### Email:

```html
<input type="email" 
       pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$"
       title="Введите корректный email">
```

### Почтовый индекс:

```html
<input type="text" 
       pattern="[0-9]{5}" 
       title="5 цифр"
       placeholder="12345">
```

### Пароль:

```html
<input type="password" 
       pattern=".{8,}" 
       title="Минимум 8 символов"
       placeholder="Пароль">
```

## Валидация

### С `required`:

```html
<input type="text" 
       pattern="[A-Za-z]{3,}" 
       required
       title="Минимум 3 буквы">
```

### Сообщение об ошибке:

Атрибут `title` используется для сообщения об ошибке, если значение не соответствует pattern.

## Заключение

**`pattern` используется для:**

1. ✅ Валидации ввода на стороне клиента
2. ✅ Проверки формата данных
3. ✅ Улучшения UX форм

**Рекомендации:**
- Используйте с `title` для сообщений об ошибках
- Комбинируйте с `required`
- Тестируйте регулярные выражения

Правильное использование `pattern` улучшает валидацию форм и пользовательский опыт.
