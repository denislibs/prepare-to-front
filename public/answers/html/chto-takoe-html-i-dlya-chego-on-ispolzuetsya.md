# Что такое HTML и для чего он используется?

HTML (HyperText Markup Language) — это стандартный язык разметки, используемый для создания и структурирования содержимого веб-страниц. HTML является фундаментальной технологией веб-разработки и служит основой для всех веб-сайтов в интернете.

## Что такое HTML?

HTML — это не язык программирования, а язык разметки, который определяет структуру и семантику веб-документов. Он использует специальные теги (tags) для обозначения различных элементов страницы, таких как заголовки, параграфы, ссылки, изображения, формы и другие компоненты.

### Основные характеристики HTML:

- ✅ **Язык разметки** — описывает структуру документа, а не его внешний вид
- ✅ **Стандартизирован** — управляется консорциумом W3C (World Wide Web Consortium)
- ✅ **Платформо-независимый** — работает на любых устройствах и операционных системах
- ✅ **Расширяемый** — постоянно развивается, добавляются новые возможности

## Для чего используется HTML?

### 1. **Создание структуры веб-страниц**

HTML определяет иерархическую структуру контента:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>
    <header>
        <h1>Главный заголовок</h1>
    </header>
    <main>
        <article>
            <h2>Заголовок статьи</h2>
            <p>Текст статьи...</p>
        </article>
    </main>
    <footer>
        <p>Подвал сайта</p>
    </footer>
</body>
</html>
```

### 2. **Семантическая разметка**

HTML5 предоставляет семантические теги, которые описывают назначение контента:

```html
<nav>Навигация</nav>
<section>Секция контента</section>
<article>Статья</article>
<aside>Боковая панель</aside>
<footer>Подвал</footer>
```

### 3. **Встраивание мультимедиа**

HTML позволяет добавлять различные типы контента:

```html
<!-- Изображения -->
<img src="photo.jpg" alt="Описание">

<!-- Видео -->
<video controls>
    <source src="video.mp4" type="video/mp4">
</video>

<!-- Аудио -->
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>
```

### 4. **Создание форм**

HTML предоставляет элементы для создания интерактивных форм:

```html
<form action="/submit" method="POST">
    <label for="name">Имя:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <button type="submit">Отправить</button>
</form>
```

### 5. **Создание ссылок и навигации**

HTML позволяет создавать гиперссылки между страницами:

```html
<a href="https://example.com">Внешняя ссылка</a>
<a href="/about.html">Внутренняя ссылка</a>
<a href="#section1">Якорная ссылка</a>
```

### 6. **Создание таблиц**

HTML предоставляет элементы для структурирования табличных данных:

```html
<table>
    <thead>
        <tr>
            <th>Имя</th>
            <th>Возраст</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Иван</td>
            <td>25</td>
        </tr>
    </tbody>
</table>
```

## Преимущества использования HTML

### 1. **Доступность**

HTML обеспечивает доступность контента для:
- Скринридеров (для людей с нарушениями зрения)
- Поисковых систем (SEO)
- Различных устройств и браузеров

### 2. **Семантика**

Семантические теги делают код более понятным и поддерживаемым:

```html
<!-- Плохо -->
<div class="header">Заголовок</div>

<!-- Хорошо -->
<header>Заголовок</header>
```

### 3. **Совместимость**

HTML работает во всех современных браузерах и на всех платформах.

### 4. **Интеграция с другими технологиями**

HTML легко интегрируется с:
- **CSS** — для стилизации
- **JavaScript** — для интерактивности
- **Серверными технологиями** — для динамического контента

## Версии HTML

### HTML 4.01 (1999)
- Стандартная версия с базовыми возможностями
- Строгий и переходный синтаксис

### XHTML (2000)
- XML-версия HTML
- Более строгий синтаксис

### HTML5 (2014)
- Современная версия с новыми семантическими тегами
- Поддержка мультимедиа
- Улучшенная поддержка форм
- API для работы с файлами, геолокацией и др.

## Структура HTML документа

```html
<!DOCTYPE html>                    <!-- Объявление типа документа -->
<html lang="ru">                   <!-- Корневой элемент -->
    <head>                         <!-- Метаданные -->
        <meta charset="UTF-8">     <!-- Кодировка -->
        <meta name="viewport" content="width=device-width">
        <title>Заголовок страницы</title>
    </head>
    <body>                         <!-- Видимое содержимое -->
        <!-- Контент страницы -->
    </body>
</html>
```

## Заключение

HTML является основой веб-разработки и используется для:
- Создания структуры веб-страниц
- Семантической разметки контента
- Встраивания мультимедиа
- Создания форм и интерактивных элементов
- Обеспечения доступности и SEO

Без HTML невозможно создать веб-страницу. Это фундаментальная технология, которая работает в связке с CSS (для стилизации) и JavaScript (для интерактивности), образуя три столпа современной веб-разработки.
