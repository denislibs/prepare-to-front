# Разница между `<canvas>` и `<svg>`?

`<canvas>` и `<svg>` — это два разных подхода к графике в HTML5. Canvas использует растровую графику через JavaScript, а SVG — векторную графику через XML-разметку.

## Основные различия

### **`<canvas>`** — Растровая графика

- Растровая графика (пиксели)
- Рисование через JavaScript API
- Высокая производительность
- Не масштабируется без потери качества
- Не доступен для скринридеров

### **`<svg>`** — Векторная графика

- Векторная графика (команды и координаты)
- Описание через XML-разметку
- Масштабируется без потери качества
- Доступен для скринридеров
- Можно стилизовать через CSS

## Сравнение

| Характеристика | Canvas | SVG |
|----------------|--------|-----|
| **Тип** | Растровый | Векторный |
| **Создание** | JavaScript | XML/CSS |
| **Масштабирование** | С потерей качества | Без потери |
| **Производительность** | Высокая | Средняя |
| **Доступность** | Нет | Да |
| **Стилизация** | Через JS | CSS |
| **Анимации** | JavaScript | CSS/SMIL |

## Когда использовать

### Canvas для:
- Игр
- Сложных анимаций
- Обработки изображений
- Графиков с большим количеством данных

### SVG для:
- Иконок и логотипов
- Простой векторной графики
- Диаграмм
- Когда нужна доступность

## Заключение

Canvas — растровая графика через JavaScript, высокая производительность. SVG — векторная графика через XML, масштабируется без потери качества и доступен для скринридеров. Выбор зависит от задачи.
