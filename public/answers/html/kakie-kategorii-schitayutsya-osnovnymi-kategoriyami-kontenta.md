# Какие категории считаются основными категориями контента?

Основные категории контента в HTML5 — это ключевые группы, которые определяют основную структуру и поведение элементов. Эти категории являются фундаментальными для понимания организации контента в HTML-документе.

## Основные категории контента

### 1. **Metadata Content (Метаданные)**

Элементы, которые предоставляют информацию о документе, но не отображаются напрямую:

**Элементы**: `<base>`, `<link>`, `<meta>`, `<noscript>`, `<script>`, `<style>`, `<title>`

```html
<head>
    <title>Заголовок страницы</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width">
    <link rel="stylesheet" href="style.css">
    <script src="script.js"></script>
</head>
```

**Характеристики**:
- Используются только в `<head>`
- Не отображаются в теле документа
- Предоставляют метаинформацию

### 2. **Flow Content (Потоковый контент)**

Большинство элементов, которые используются в теле документа и участвуют в потоке документа:

**Элементы**: Почти все элементы, включая `<p>`, `<div>`, `<span>`, `<a>`, `<img>`, `<form>`, `<table>`, `<ul>`, `<ol>`, `<h1>`-`<h6>`, `<article>`, `<section>`, `<nav>`, `<aside>`, `<header>`, `<footer>`, `<main>` и многие другие.

```html
<body>
    <header>Шапка</header>
    <main>
        <article>
            <h1>Заголовок</h1>
            <p>Параграф текста</p>
            <div>Блок контента</div>
        </article>
    </main>
    <footer>Подвал</footer>
</body>
```

**Характеристики**:
- Используются в `<body>`
- Участвуют в потоке документа
- Могут содержать другие flow элементы

### 3. **Sectioning Content (Секционирующий контент)**

Элементы, которые создают секции в структуре документа и влияют на структуру заголовков:

**Элементы**: `<article>`, `<aside>`, `<nav>`, `<section>`

```html
<article>
    <h1>Статья</h1>
    <p>Контент статьи</p>
</article>

<section>
    <h2>Секция</h2>
    <p>Контент секции</p>
</section>

<nav>
    <ul>
        <li><a href="#">Навигация</a></li>
    </ul>
</nav>

<aside>
    <p>Боковая панель</p>
</aside>
```

**Характеристики**:
- Создают логические секции документа
- Влияют на структуру заголовков (outline algorithm)
- Улучшают семантику документа

### 4. **Heading Content (Заголовочный контент)**

Элементы заголовков, которые определяют структуру документа:

**Элементы**: `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`, `<hgroup>`

```html
<h1>Главный заголовок</h1>
<h2>Заголовок уровня 2</h2>
<h3>Заголовок уровня 3</h3>
```

**Характеристики**:
- Определяют иерархию документа
- Используются для создания структуры
- Важны для SEO и доступности

### 5. **Phrasing Content (Фразовый контент)**

Элементы, которые размечают текст внутри параграфов и других текстовых элементов:

**Элементы**: `<a>`, `<abbr>`, `<b>`, `<bdi>`, `<bdo>`, `<br>`, `<button>`, `<cite>`, `<code>`, `<data>`, `<dfn>`, `<em>`, `<i>`, `<input>`, `<kbd>`, `<label>`, `<mark>`, `<output>`, `<q>`, `<ruby>`, `<s>`, `<samp>`, `<select>`, `<small>`, `<span>`, `<strong>`, `<sub>`, `<sup>`, `<textarea>`, `<time>`, `<u>`, `<var>`, `<wbr>`

```html
<p>
    Это <strong>жирный</strong> текст с 
    <em>курсивом</em>, <a href="#">ссылкой</a>,
    <code>кодом</code> и <mark>выделением</mark>.
</p>
```

**Характеристики**:
- Используются внутри текстовых элементов
- Не создают блочные структуры
- Размечают текст семантически

### 6. **Embedded Content (Встроенный контент)**

Элементы, которые импортируют другой ресурс или встраивают контент:

**Элементы**: `<audio>`, `<canvas>`, `<embed>`, `<iframe>`, `<img>`, `<math>`, `<object>`, `<picture>`, `<svg>`, `<video>`

```html
<img src="photo.jpg" alt="Фото">
<video src="video.mp4" controls></video>
<audio src="audio.mp3" controls></audio>
<iframe src="page.html"></iframe>
<canvas width="800" height="600"></canvas>
<svg>...</svg>
```

**Характеристики**:
- Импортируют внешние ресурсы
- Могут быть интерактивными
- Требуют атрибутов для источника

### 7. **Interactive Content (Интерактивный контент)**

Элементы, с которыми пользователь может взаимодействовать:

**Элементы**: `<a>`, `<button>`, `<details>`, `<embed>`, `<iframe>`, `<input>`, `<label>`, `<select>`, `<textarea>`

```html
<a href="#">Ссылка</a>
<button>Кнопка</button>
<input type="text">
<select>
    <option>Вариант</option>
</select>
<details>
    <summary>Раскрыть</summary>
    <p>Скрытый контент</p>
</details>
```

**Характеристики**:
- Позволяют взаимодействие с пользователем
- Могут получать фокус
- Обрабатывают события

## Иерархия категорий

```
Metadata Content
    ↓
Flow Content (основная категория)
    ├── Sectioning Content
    ├── Heading Content
    ├── Phrasing Content
    ├── Embedded Content
    └── Interactive Content
```

## Практическое применение

### Структура документа с основными категориями:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <!-- Metadata Content -->
    <meta charset="UTF-8">
    <title>Документ</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- Flow Content -->
    <header>
        <!-- Sectioning Content -->
        <nav>
            <!-- Flow Content -->
            <ul>
                <li><a href="#">Главная</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <!-- Sectioning Content -->
        <article>
            <!-- Heading Content -->
            <h1>Заголовок</h1>
            
            <!-- Flow Content -->
            <p>
                <!-- Phrasing Content -->
                Текст с <strong>выделением</strong>
            </p>
            
            <!-- Embedded Content -->
            <img src="photo.jpg" alt="Фото">
        </article>
    </main>
</body>
</html>
```

## Заключение

Основные категории контента в HTML5:
1. **Metadata** — информация о документе
2. **Flow** — основной поток контента
3. **Sectioning** — секции документа
4. **Heading** — заголовки
5. **Phrasing** — текстовая разметка
6. **Embedded** — встроенный контент
7. **Interactive** — интерактивные элементы

Эти категории определяют:
- Правила вложенности элементов
- Допустимое использование элементов
- Семантическую структуру документа
- Поведение элементов в документе

Понимание основных категорий контента помогает создавать валидную, семантически правильную и доступную HTML-разметку.
