# Какие HTML-элементы используются для создания и форматирования таблиц? Какие атрибуты у них есть?

HTML-таблицы создаются с помощью набора семантических элементов, каждый из которых имеет свое назначение и атрибуты для форматирования и структурирования данных.

## Основные элементы таблиц

### `<table>` — Контейнер таблицы

```html
<table>
    <!-- Содержимое таблицы -->
</table>
```

### `<thead>` — Заголовок таблицы

```html
<thead>
    <tr>
        <th>Заголовок 1</th>
        <th>Заголовок 2</th>
    </tr>
</thead>
```

### `<tbody>` — Тело таблицы

```html
<tbody>
    <tr>
        <td>Данные 1</td>
        <td>Данные 2</td>
    </tr>
</tbody>
```

### `<tfoot>` — Подвал таблицы

```html
<tfoot>
    <tr>
        <td>Итого</td>
        <td>100</td>
    </tr>
</tfoot>
```

### `<tr>` — Строка таблицы

```html
<tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2</td>
</tr>
```

### `<th>` — Заголовочная ячейка

```html
<th>Заголовок</th>
```

### `<td>` — Ячейка с данными

```html
<td>Данные</td>
```

## Атрибуты элементов

### Атрибуты `<table>`

#### `border` — Граница

```html
<table border="1">
```

#### `cellpadding` — Отступы в ячейках

```html
<table cellpadding="10">
```

#### `cellspacing` — Расстояние между ячейками

```html
<table cellspacing="5">
```

**Примечание:** Эти атрибуты устарели, используйте CSS.

### Атрибуты `<th>` и `<td>`

#### `colspan` — Объединение столбцов

```html
<th colspan="2">Объединенный заголовок</th>
```

#### `rowspan` — Объединение строк

```html
<td rowspan="2">Объединенная ячейка</td>
```

#### `scope` — Область действия (для `<th>`)

```html
<th scope="col">Заголовок столбца</th>
<th scope="row">Заголовок строки</th>
```

#### `headers` — Связь с заголовками

```html
<td headers="header1 header2">Данные</td>
```

#### `abbr` — Сокращение

```html
<th abbr="ID">Идентификатор</th>
```

## Примеры использования

### Базовая таблица:

```html
<table>
    <thead>
        <tr>
            <th>Имя</th>
            <th>Возраст</th>
            <th>Город</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Иван</td>
            <td>25</td>
            <td>Москва</td>
        </tr>
        <tr>
            <td>Мария</td>
            <td>30</td>
            <td>Санкт-Петербург</td>
        </tr>
    </tbody>
</table>
```

### Таблица с объединением ячеек:

```html
<table>
    <thead>
        <tr>
            <th colspan="2">Имя</th>
            <th>Возраст</th>
        </tr>
        <tr>
            <th>Имя</th>
            <th>Фамилия</th>
            <th>Возраст</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Иван</td>
            <td>Иванов</td>
            <td>25</td>
        </tr>
    </tbody>
</table>
```

### Таблица с подвалом:

```html
<table>
    <thead>
        <tr>
            <th>Товар</th>
            <th>Цена</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Товар 1</td>
            <td>100</td>
        </tr>
        <tr>
            <td>Товар 2</td>
            <td>200</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th>Итого</th>
            <td>300</td>
        </tr>
    </tfoot>
</table>
```

### Таблица с `scope`:

```html
<table>
    <thead>
        <tr>
            <th scope="col">Товар</th>
            <th scope="col">Цена</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th scope="row">Товар 1</th>
            <td>100</td>
        </tr>
    </tbody>
</table>
```

## Дополнительные элементы

### `<caption>` — Заголовок таблицы

```html
<table>
    <caption>Список товаров</caption>
    <!-- Содержимое -->
</table>
```

### `<colgroup>` и `<col>` — Группировка столбцов

```html
<table>
    <colgroup>
        <col style="width: 50%">
        <col style="width: 50%">
    </colgroup>
    <!-- Содержимое -->
</table>
```

## Заключение

**Элементы таблиц:**

1. ✅ `<table>` — контейнер
2. ✅ `<thead>`, `<tbody>`, `<tfoot>` — секции
3. ✅ `<tr>` — строка
4. ✅ `<th>` — заголовочная ячейка
5. ✅ `<td>` — ячейка с данными
6. ✅ `<caption>` — заголовок
7. ✅ `<colgroup>`, `<col>` — группировка столбцов

**Атрибуты:**

- `colspan` — объединение столбцов
- `rowspan` — объединение строк
- `scope` — область действия
- `headers` — связь с заголовками

**Рекомендации:**
- Используйте семантические элементы
- Используйте `scope` для доступности
- Используйте CSS для стилизации
- Избегайте устаревших атрибутов (`border`, `cellpadding`, `cellspacing`)

Правильное использование элементов и атрибутов таблиц создает доступные и структурированные таблицы.
