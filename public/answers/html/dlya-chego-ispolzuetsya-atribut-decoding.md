# Для чего используется атрибут `decoding`?

Атрибут `decoding` в HTML указывает браузеру, как декодировать изображение: синхронно или асинхронно. Это влияет на производительность и отображение страницы.

## Что такое decoding?

`decoding` — это атрибут элемента `<img>`, который указывает стратегию декодирования изображения.

## Значения

### `sync` — Синхронное декодирование

```html
<img src="photo.jpg" decoding="sync" alt="Фото">
```

- Декодирование происходит синхронно
- Блокирует рендеринг до завершения
- Используется для критичных изображений

### `async` — Асинхронное декодирование

```html
<img src="photo.jpg" decoding="async" alt="Фото">
```

- Декодирование происходит асинхронно
- Не блокирует рендеринг
- Улучшает производительность
- Рекомендуется для большинства случаев

### `auto` — Автоматическое (по умолчанию)

```html
<img src="photo.jpg" decoding="auto" alt="Фото">
```

- Браузер выбирает стратегию
- Поведение по умолчанию

## Примеры использования

### Критичное изображение:

```html
<img src="hero.jpg" decoding="sync" alt="Главное изображение">
```

### Обычное изображение:

```html
<img src="photo.jpg" decoding="async" alt="Фото">
```

### С другими атрибутами:

```html
<img src="photo.jpg" 
     decoding="async" 
     loading="lazy" 
     alt="Фото">
```

## Влияние на производительность

### `decoding="async"`:

- Не блокирует рендеринг
- Улучшает время загрузки страницы
- Изображение может появиться позже

### `decoding="sync"`:

- Блокирует рендеринг
- Изображение появляется сразу
- Может замедлить загрузку

## Заключение

**`decoding` используется для:**

1. ✅ Контроля декодирования изображений
2. ✅ Улучшения производительности
3. ✅ Оптимизации рендеринга

**Значения:**
- `sync` — синхронное (для критичных изображений)
- `async` — асинхронное (рекомендуется)
- `auto` — автоматическое (по умолчанию)

**Рекомендации:**
- Используйте `async` для большинства изображений
- Используйте `sync` для критичных изображений
- Комбинируйте с `loading="lazy"` для оптимизации

Правильное использование `decoding` улучшает производительность загрузки страницы.
