# Для чего используются тэги `<tr>`, `<th>`, `<td>`?

Теги `<tr>`, `<th>` и `<td>` используются для создания и структурирования таблиц в HTML. Каждый тег имеет свое назначение в структуре таблицы.

## Назначение тегов

### 1. **`<tr>`** — Table Row (Строка таблицы)

Определяет строку в таблице.

```html
<tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2</td>
</tr>
```

**Характеристики:**
- Содержит ячейки (`<td>` или `<th>`)
- Может содержать только ячейки
- Не может содержать другие `<tr>`

### 2. **`<th>`** — Table Header (Заголовок таблицы)

Определяет заголовочную ячейку.

```html
<th>Заголовок</th>
```

**Характеристики:**
- Используется для заголовков столбцов или строк
- По умолчанию жирный и центрированный
- Семантически указывает на заголовок

### 3. **`<td>`** — Table Data (Данные таблицы)

Определяет обычную ячейку с данными.

```html
<td>Данные</td>
```

**Характеристики:**
- Содержит данные таблицы
- Обычный текст, не жирный
- Выравнивание по умолчанию слева

## Структура таблицы

### Базовая таблица:

```html
<table>
    <thead>
        <tr>
            <th>Имя</th>
            <th>Возраст</th>
            <th>Город</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Иван</td>
            <td>25</td>
            <td>Москва</td>
        </tr>
        <tr>
            <td>Мария</td>
            <td>30</td>
            <td>Санкт-Петербург</td>
        </tr>
    </tbody>
</table>
```

## Атрибуты

### Атрибуты `<th>` и `<td>`:

#### `colspan` — Объединение столбцов:

```html
<tr>
    <th colspan="2">Объединенный заголовок</th>
</tr>
<tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2</td>
</tr>
```

#### `rowspan` — Объединение строк:

```html
<tr>
    <td rowspan="2">Объединенная ячейка</td>
    <td>Ячейка 1</td>
</tr>
<tr>
    <td>Ячейка 2</td>
</tr>
```

#### `scope` — Область действия (для `<th>`):

```html
<th scope="col">Заголовок столбца</th>
<th scope="row">Заголовок строки</th>
```

#### `headers` — Связь с заголовками:

```html
<td headers="name-header">Иван</td>
```

## Примеры использования

### Простая таблица:

```html
<table>
    <tr>
        <th>Продукт</th>
        <th>Цена</th>
        <th>Количество</th>
    </tr>
    <tr>
        <td>Хлеб</td>
        <td>50 руб</td>
        <td>2</td>
    </tr>
    <tr>
        <td>Молоко</td>
        <td>80 руб</td>
        <td>1</td>
    </tr>
</table>
```

### Таблица с объединенными ячейками:

```html
<table>
    <tr>
        <th colspan="2">Персональные данные</th>
        <th>Контакты</th>
    </tr>
    <tr>
        <th>Имя</th>
        <th>Возраст</th>
        <th>Email</th>
    </tr>
    <tr>
        <td>Иван</td>
        <td>25</td>
        <td rowspan="2">ivan@example.com</td>
    </tr>
    <tr>
        <td>Мария</td>
        <td>30</td>
    </tr>
</table>
```

### Семантическая таблица:

```html
<table>
    <caption>Список сотрудников</caption>
    <thead>
        <tr>
            <th scope="col">ID</th>
            <th scope="col">Имя</th>
            <th scope="col">Должность</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>1</td>
            <td>Иван</td>
            <td>Разработчик</td>
        </tr>
        <tr>
            <td>2</td>
            <td>Мария</td>
            <td>Дизайнер</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Всего: 2 сотрудника</td>
        </tr>
    </tfoot>
</table>
```

## Стилизация

### Базовые стили:

```css
table {
    border-collapse: collapse;
    width: 100%;
}

th, td {
    border: 1px solid #ddd;
    padding: 8px;
    text-align: left;
}

th {
    background-color: #f2f2f2;
    font-weight: bold;
}

tr:nth-child(even) {
    background-color: #f9f9f9;
}
```

## Доступность

### Правильная доступность:

```html
<table>
    <caption>Описание таблицы</caption>
    <thead>
        <tr>
            <th scope="col">Заголовок 1</th>
            <th scope="col">Заголовок 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th scope="row">Заголовок строки</th>
            <td>Данные</td>
        </tr>
    </tbody>
</table>
```

## Заключение

**Назначение тегов:**

1. **`<tr>`** — строка таблицы
   - Содержит ячейки
   - Группирует ячейки в строки

2. **`<th>`** — заголовочная ячейка
   - Для заголовков столбцов/строк
   - Семантически указывает на заголовок
   - По умолчанию жирный и центрированный

3. **`<td>`** — ячейка с данными
   - Содержит данные таблицы
   - Обычное форматирование

**Ключевые моменты:**
- ✅ Используйте `<th>` для заголовков
- ✅ Используйте `<td>` для данных
- ✅ Используйте `scope` для доступности
- ✅ Используйте `colspan` и `rowspan` для объединения ячеек

Правильное использование этих тегов создает структурированные, доступные и семантически правильные таблицы.
