# Расскажите о тэге `<samp>`?

Тэг `<samp>` в HTML используется для обозначения вывода программы или системы. Он представляет пример вывода, который пользователь видит на экране.

## Что такое `<samp>`?

`<samp>` — это семантический HTML-элемент, который обозначает пример вывода программы, скрипта или системы.

## Использование

### Базовый синтаксис:

```html
<samp>Пример вывода</samp>
```

## Примеры использования

### Вывод программы:

```html
<p>Результат выполнения программы:</p>
<samp>Hello, World!</samp>
```

### Вывод команды:

```html
<p>Вывод команды:</p>
<samp>$ npm install</samp>
<samp>npm WARN deprecated package@1.0.0</samp>
```

### Сообщения об ошибках:

```html
<p>Ошибка:</p>
<samp>Error: Cannot read property 'name' of undefined</samp>
```

### Вывод консоли:

```html
<pre><samp>
> node app.js
Server running on port 3000
Connected to database
</samp></pre>
```

## Стилизация

### По умолчанию:

Браузеры обычно отображают `<samp>` моноширинным шрифтом.

### Кастомная стилизация:

```css
samp {
    font-family: 'Courier New', monospace;
    background-color: #f4f4f4;
    padding: 0.2rem 0.4rem;
    border-radius: 3px;
    color: #333;
}
```

## Семантика

### Доступность:

`<samp>` помогает скринридерам понять, что текст является примером вывода программы.

### SEO:

Поисковые системы понимают, что содержимое `<samp>` — это пример вывода, а не обычный текст.

## Сравнение с другими тегами

### `<samp>` vs `<code>`:

```html
<!-- <code> — код программы -->
<code>const x = 10;</code>

<!-- <samp> — вывод программы -->
<samp>10</samp>
```

### `<samp>` vs `<kbd>`:

```html
<!-- <kbd> — ввод пользователя -->
<kbd>Ctrl + C</kbd>

<!-- <samp> — вывод системы -->
<samp>Process terminated</samp>
```

### `<samp>` vs `<pre>`:

```html
<!-- <pre> — предформатированный текст -->
<pre>Многострочный
текст</pre>

<!-- <samp> — пример вывода -->
<samp>Пример вывода</samp>
```

## Комбинирование

### С `<pre>`:

```html
<pre><samp>
$ git status
On branch main
Changes not staged for commit
</samp></pre>
```

### С `<code>`:

```html
<p>Выполните команду:</p>
<code>npm install</code>
<p>Вы увидите:</p>
<samp>package installed successfully</samp>
```
