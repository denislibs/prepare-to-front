# Чем отличается `<article>` от `<section>`?

`<article>` и `<section>` — это семантические элементы HTML5 для структурирования контента, но они имеют разные назначения: `<article>` для независимого контента, а `<section>` для тематических групп контента.

## Разница между `<article>` и `<section>`

### **`<article>`** — Независимый контент

Используется для самодостаточного, независимого контента, который может быть переиспользован.

**Характеристики:**
- ✅ Независимый и самодостаточный
- ✅ Может быть переиспользован (RSS, блог-пост)
- ✅ Имеет смысл вне контекста страницы
- ✅ Обычно имеет заголовок

### **`<section>`** — Тематическая группа

Используется для тематической группы связанного контента.

**Характеристики:**
- ✅ Тематическая группа контента
- ✅ Обычно имеет заголовок
- ✅ Часть более крупного целого
- ✅ Не обязательно независимый

## Примеры использования

### `<article>` — Статья:

```html
<article>
    <header>
        <h2>Заголовок статьи</h2>
        <time datetime="2024-01-15">15 января 2024</time>
    </header>
    <p>Содержимое статьи...</p>
    <footer>
        <p>Автор: Иван Иванов</p>
    </footer>
</article>
```

### `<section>` — Секция:

```html
<section>
    <h2>Введение</h2>
    <p>Текст введения...</p>
</section>

<section>
    <h2>Основной контент</h2>
    <p>Основной текст...</p>
</section>
```

## Вложенность

### Article может содержать sections:

```html
<article>
    <header>
        <h1>Заголовок статьи</h1>
    </header>
    
    <section>
        <h2>Введение</h2>
        <p>Текст...</p>
    </section>
    
    <section>
        <h2>Основная часть</h2>
        <p>Текст...</p>
    </section>
</article>
```

### Section может содержать articles:

```html
<section>
    <h2>Последние новости</h2>
    
    <article>
        <h3>Новость 1</h3>
        <p>Текст новости...</p>
    </article>
    
    <article>
        <h3>Новость 2</h3>
        <p>Текст новости...</p>
    </article>
</section>
```

## Сравнение

| Характеристика | `<article>` | `<section>` |
|----------------|-------------|-------------|
| **Назначение** | Независимый контент | Тематическая группа |
| **Независимость** | Да | Нет |
| **Переиспользование** | Да (RSS) | Нет |
| **Заголовок** | Обычно есть | Обычно есть |

## Заключение

**Разница:**

- **`<article>`**: Независимый, самодостаточный контент (статьи, посты, новости)
- **`<section>`**: Тематическая группа контента (секции, разделы)

**Использование:**
- `<article>` для статей, постов, новостей
- `<section>` для тематических групп контента

**Вложенность:**
- Article может содержать sections
- Section может содержать articles

Правильный выбор элемента улучшает семантику и структуру документа.
