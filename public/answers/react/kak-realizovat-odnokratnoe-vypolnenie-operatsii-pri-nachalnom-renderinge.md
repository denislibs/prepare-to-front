# Как реализовать однократное выполнение операции при начальном рендеринге?

Для однократного выполнения операции при начальном рендеринге в React используется `useEffect` с пустым массивом зависимостей. Это эквивалент `componentDidMount` в функциональных компонентах.

## Реализация

### С useEffect:

```jsx
function Component() {
    useEffect(() => {
        // Выполнится только один раз при монтировании
        fetchData();
    }, []); // Пустой массив = один раз
    
    return <div>Component</div>;
}
```

### Пример загрузки данных:

```jsx
function UserProfile({ userId }) {
    const [user, setUser] = useState(null);
    
    useEffect(() => {
        fetchUser(userId).then(setUser);
    }, []); // Только при монтировании
    
    if (!user) return <div>Загрузка...</div>;
    return <div>{user.name}</div>;
}
```

## Заключение

**Однократное выполнение:**

1. ✅ useEffect с пустым массивом зависимостей
2. ✅ Эквивалент componentDidMount
3. ✅ Выполняется при монтировании

**Рекомендации:**

- Используйте пустой массив для однократного выполнения
- Очищайте ресурсы в return функции
- Понимайте зависимости

useEffect с пустым массивом — стандартный способ однократного выполнения операций.
