# Техники оптимизации перфоманса React?

Оптимизация производительности React включает множество техник: мемоизацию, ленивую загрузку, виртуализацию, код-сплиттинг и другие методы.

## Основные техники

### 1. **Мемоизация компонентов**

```jsx
const MemoizedComponent = React.memo(Component);
```

### 2. **Мемоизация значений**

```jsx
const value = useMemo(() => expensiveCalculation(a, b), [a, b]);
```

### 3. **Мемоизация функций**

```jsx
const callback = useCallback(() => doSomething(a, b), [a, b]);
```

### 4. **Ленивая загрузка**

```jsx
const LazyComponent = React.lazy(() => import('./Component'));
```

### 5. **Виртуализация списков**

```jsx
import { FixedSizeList } from 'react-window';
```

### 6. **Код-сплиттинг**

```jsx
const Component = lazy(() => import('./Component'));
```

### 7. **Оптимизация изображений**

```jsx
<img src={image} loading="lazy" />
```
