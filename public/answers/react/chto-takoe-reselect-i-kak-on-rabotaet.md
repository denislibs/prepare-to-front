# Что такое Reselect и как он работает?

Reselect — это библиотека для создания мемоизированных селекторов в Redux. Она позволяет создавать селекторы, которые пересчитываются только при изменении входных данных, что оптимизирует производительность.

## Что такое Reselect?

Reselect создает мемоизированные селекторы, которые кэшируют результаты вычислений и пересчитываются только при изменении зависимостей.

## Использование

### Базовый пример:

```jsx
import { createSelector } from 'reselect';

const getTodos = (state) => state.todos;
const getFilter = (state) => state.filter;

const getFilteredTodos = createSelector(
    [getTodos, getFilter],
    (todos, filter) => {
        return todos.filter(todo => todo.category === filter);
    }
);
```

## Преимущества

### 1. **Мемоизация**

Результаты кэшируются и пересчитываются только при изменении зависимостей.

### 2. **Производительность**

Предотвращает ненужные пересчеты.

### 3. **Композиция**

Селекторы можно комбинировать.
