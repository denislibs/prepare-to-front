# Что такое "Hydration" в контексте серверного-рендеренга React-приложений?

Hydration (гидратация) — это процесс "оживления" статического HTML, отрендеренного на сервере, путем подключения React к существующему DOM и добавления интерактивности.

## Что такое Hydration?

Hydration — это процесс, когда React "оживляет" статический HTML, добавляя обработчики событий и делая компоненты интерактивными.

## Процесс

### 1. **Серверный рендеринг**

```jsx
// На сервере
const html = ReactDOMServer.renderToString(<App />);
```

### 2. **Отправка HTML**

```html
<div id="root"><!-- Статический HTML --></div>
<script src="/bundle.js"></script>
```

### 3. **Гидратация на клиенте**

```jsx
// На клиенте
ReactDOM.hydrateRoot(document.getElementById('root'), <App />);
```

## Важно

### Совпадение HTML:

HTML на сервере и клиенте должен совпадать, иначе возникнут ошибки.
