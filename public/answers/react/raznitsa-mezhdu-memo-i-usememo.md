# Разница между `memo` и `useMemo`?

`memo` и `useMemo` — это разные инструменты оптимизации в React. `memo` мемоизирует компонент, а `useMemo` мемоизирует значение.

## React.memo

### Что это?

`memo` — это HOC (Higher-Order Component), который мемоизирует компонент, предотвращая перерисовку при неизменных пропсах.

### Использование:

```jsx
const MemoizedComponent = React.memo(function Component({ name }) {
    return <div>{name}</div>;
});
```

## useMemo

### Что это?

`useMemo` — это хук, который мемоизирует результат вычисления.

### Использование:

```jsx
function Component({ items }) {
    const filtered = useMemo(() => {
        return items.filter(item => item.active);
    }, [items]);
    
    return <div>{filtered.map(item => <Item key={item.id} item={item} />)}</div>;
}
```

## Различия

### memo:

- Мемоизирует компонент
- Предотвращает перерисовку
- Для компонентов

### useMemo:

- Мемоизирует значение
- Оптимизирует вычисления
- Для значений
