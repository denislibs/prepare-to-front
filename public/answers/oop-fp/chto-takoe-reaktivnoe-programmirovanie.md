# Что такое реактивное программирование?

Реактивное программирование — это парадигма программирования, ориентированная на потоки данных и распространение изменений. В реактивном программировании программа автоматически реагирует на изменения в данных.

## Что такое реактивное программирование?

Реактивное программирование — это программирование с асинхронными потоками данных, где изменения автоматически распространяются через систему.

## Основные концепции

### 1. **Потоки данных (Streams)**

Последовательности событий во времени.

```javascript
// RxJS пример
import { fromEvent } from 'rxjs';

const clicks = fromEvent(document, 'click');
clicks.subscribe(event => console.log('Клик!'));
```

### 2. **Observable**

Представление потока данных.

```javascript
import { Observable } from 'rxjs';

const observable = new Observable(subscriber => {
    subscriber.next(1);
    subscriber.next(2);
    subscriber.next(3);
    subscriber.complete();
});
```

### 3. **Операторы**

Трансформация потоков.

```javascript
import { map, filter } from 'rxjs/operators';

numbers.pipe(
    filter(x => x > 2),
    map(x => x * 2)
).subscribe(console.log);
```

## Преимущества

### 1. **Асинхронность**

Легко работать с асинхронными данными.

### 2. **Композиция**

Легко комбинировать потоки.

### 3. **Реактивность**

Автоматическое обновление при изменениях.
