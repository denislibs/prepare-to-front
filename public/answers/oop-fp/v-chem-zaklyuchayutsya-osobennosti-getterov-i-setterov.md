# В чем заключаются особенности геттеров и сеттеров?

Геттеры и сеттеры — это специальные методы в JavaScript, которые позволяют определять, как свойства объекта читаются и записываются. Они предоставляют контроль над доступом к свойствам.

## Геттеры (Getters)

### Особенности:

- Вызываются при чтении свойства
- Не принимают параметры
- Могут вычислять значение динамически

```javascript
const obj = {
    _value: 10,
    
    get value() {
        return this._value * 2;
    }
};

console.log(obj.value); // 20
```

## Сеттеры (Setters)

### Особенности:

- Вызываются при записи свойства
- Принимают один параметр
- Могут валидировать значения

```javascript
const obj = {
    _age: 0,
    
    set age(value) {
        if (value > 0 && value < 150) {
            this._age = value;
        } else {
            throw new Error('Некорректный возраст');
        }
    },
    
    get age() {
        return this._age;
    }
};

obj.age = 25; // OK
obj.age = 200; // Error
```

## Преимущества

### 1. **Валидация**

Проверка значений при установке.

### 2. **Вычисляемые свойства**

Динамическое вычисление значений.

### 3. **Инкапсуляция**

Скрытие внутренней реализации.

## Заключение

**Геттеры:**

1. ✅ Чтение свойств
2. ✅ Вычисляемые значения
3. ✅ Без параметров

**Сеттеры:**

1. ✅ Запись свойств
2. ✅ Валидация
3. ✅ Один параметр

**Рекомендации:**

- Используйте для валидации
- Применяйте для вычисляемых свойств
- Используйте для инкапсуляции

Геттеры и сеттеры — мощный инструмент для контроля доступа к свойствам.
