# Разница между ООП и ФП в JavaScript?

ООП (Объектно-Ориентированное Программирование) и ФП (Функциональное Программирование) — это две разные парадигмы программирования. JavaScript поддерживает обе парадигмы, и понимание различий важно для выбора правильного подхода.

## ООП в JavaScript

### Характеристики:

- Основано на объектах и классах
- Использует состояние и методы
- Инкапсуляция данных
- Наследование и полиморфизм

```javascript
class User {
    constructor(name) {
        this.name = name;
    }
    
    greet() {
        return `Привет, ${this.name}!`;
    }
}

const user = new User('Иван');
user.greet(); // "Привет, Иван!"
```

## ФП в JavaScript

### Характеристики:

- Основано на функциях
- Избегает изменяемого состояния
- Чистые функции
- Неизменяемость данных

```javascript
const greet = (name) => `Привет, ${name}!`;

const users = ['Иван', 'Петр'];
const greetings = users.map(greet);
// ["Привет, Иван!", "Привет, Петр!"]
```

## Различия

| Характеристика | ООП | ФП |
|----------------|-----|-----|
| Основа | Объекты/классы | Функции |
| Состояние | Изменяемое | Неизменяемое |
| Фокус | Данные и поведение | Вычисления |
| Побочные эффекты | Допускаются | Избегаются |

## Когда использовать

### ООП:

- ✅ Сложные объекты с состоянием
- ✅ Моделирование реального мира
- ✅ Инкапсуляция данных

### ФП:

- ✅ Обработка данных
- ✅ Трансформации
- ✅ Предсказуемость

## Заключение

**ООП:**

1. ✅ Объекты и классы
2. ✅ Состояние и методы
3. ✅ Инкапсуляция

**ФП:**

1. ✅ Функции
2. ✅ Неизменяемость
3. ✅ Чистые функции

**Рекомендации:**

- Используйте ООП для моделирования
- Используйте ФП для трансформаций
- Комбинируйте подходы при необходимости

JavaScript позволяет использовать обе парадигмы эффективно.
