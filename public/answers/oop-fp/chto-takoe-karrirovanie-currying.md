# Что такое каррирование (Currying)?

Каррирование (Currying) — это техника функционального программирования, при которой функция с несколькими аргументами преобразуется в последовательность функций, каждая из которых принимает один аргумент. Это позволяет создавать более гибкие и переиспользуемые функции.

## Что такое каррирование?

Каррирование — это процесс преобразования функции с несколькими аргументами в цепочку функций, каждая из которых принимает один аргумент.

## Примеры

### Обычная функция:

```javascript
function add(a, b, c) {
    return a + b + c;
}

add(1, 2, 3); // 6
```

### Каррированная функция:

```javascript
function add(a) {
    return function(b) {
        return function(c) {
            return a + b + c;
        };
    };
}

add(1)(2)(3); // 6
```

### Современный синтаксис:

```javascript
const add = a => b => c => a + b + c;

add(1)(2)(3); // 6
```

## Практическое применение

### Частичное применение:

```javascript
const multiply = a => b => a * b;

const double = multiply(2);
double(5); // 10

const triple = multiply(3);
triple(5); // 15
```

### Композиция функций:

```javascript
const pipe = (...fns) => x => fns.reduce((v, f) => f(v), x);

const add = a => b => a + b;
const multiply = a => b => a * b;

const addThenMultiply = pipe(
    add(5),
    multiply(2)
);

addThenMultiply(3); // 16
```

## Преимущества

### 1. **Переиспользование**

Создание специализированных функций.

### 2. **Гибкость**

Частичное применение аргументов.

### 3. **Композиция**

Легче комбинировать функции.

## Заключение

**Каррирование — это:**

1. ✅ Преобразование функции с несколькими аргументами
2. ✅ Цепочка функций с одним аргументом
3. ✅ Частичное применение

**Преимущества:**

- Переиспользование
- Гибкость
- Композиция

**Рекомендации:**

- Используйте для создания специализированных функций
- Применяйте для частичного применения
- Комбинируйте с композицией

Каррирование — мощная техника функционального программирования.
