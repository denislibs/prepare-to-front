# Что такое композиция в контексте JavaScript?

Композиция в JavaScript — это техника создания сложных объектов путем объединения более простых объектов. Композиция позволяет создавать гибкие и переиспользуемые структуры без жесткой иерархии наследования.

## Что такое композиция?

Композиция — это способ организации кода, при котором объекты содержат другие объекты, создавая отношения "имеет" вместо "является".

## Примеры

### Композиция объектов:

```javascript
class Engine {
    start() {
        console.log('Двигатель запущен');
    }
}

class Wheels {
    rotate() {
        console.log('Колеса вращаются');
    }
}

class Car {
    constructor() {
        this.engine = new Engine();
        this.wheels = new Wheels();
    }
    
    start() {
        this.engine.start();
        this.wheels.rotate();
    }
}
```

### Композиция функций:

```javascript
const compose = (...fns) => x => fns.reduceRight((v, f) => f(v), x);

const add = x => y => x + y;
const multiply = x => y => x * y;

const addThenMultiply = compose(
    multiply(2),
    add(5)
);

addThenMultiply(3); // 16
```

## Преимущества

### 1. **Гибкость**

Легко изменять и заменять компоненты.

### 2. **Переиспользование**

Компоненты можно переиспользовать.

### 3. **Тестируемость**

Легче тестировать отдельные компоненты.

## Заключение

**Композиция — это:**

1. ✅ Объединение объектов
2. ✅ Отношение "имеет"
3. ✅ Гибкость и переиспользование

**Преимущества:**

- Гибкость
- Переиспользование
- Тестируемость

**Рекомендации:**

- Применяйте для композиции функций
- Предпочитайте композицию наследованию

Композиция — мощный инструмент для организации кода в JavaScript.
