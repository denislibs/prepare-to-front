# Разница между функцией и методом?

Функция и метод — это связанные, но разные понятия в программировании. Понимание различий важно для работы с объектно-ориентированным и функциональным программированием.

## Функция

### Характеристики:

- Независимая единица кода
- Не привязана к объекту
- Вызывается напрямую

```javascript
// Функция
function add(a, b) {
    return a + b;
}

add(2, 3); // 5
```

## Метод

### Характеристики:

- Принадлежит объекту
- Вызывается через объект
- Имеет доступ к `this`

```javascript
// Метод
const calculator = {
    add(a, b) {
        return a + b;
    }
};

calculator.add(2, 3); // 5
```

## Различия

| Характеристика | Функция | Метод |
|----------------|---------|-------|
| Привязка | Нет | К объекту |
| Вызов | Напрямую | Через объект |
| `this` | Нет (или глобальный) | Указывает на объект |

## Заключение

**Функция:**

1. ✅ Независимая
2. ✅ Вызывается напрямую
3. ❌ Нет доступа к `this` объекта

**Метод:**

1. ✅ Принадлежит объекту
2. ✅ Вызывается через объект
3. ✅ Имеет доступ к `this`

**Рекомендации:**

- Используйте функции для независимой логики
- Используйте методы для логики объекта
- Понимайте контекст `this`

Понимание различий важно для правильного использования.
