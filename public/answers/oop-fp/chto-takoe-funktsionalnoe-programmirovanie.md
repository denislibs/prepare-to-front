# Что такое функциональное программирование?

Функциональное программирование (ФП) — это парадигма программирования, которая рассматривает вычисления как оценку математических функций и избегает изменения состояния и изменяемых данных. ФП делает акцент на использовании чистых функций и неизменяемости.

## Что такое функциональное программирование?

Функциональное программирование — это стиль программирования, основанный на использовании функций как основных строительных блоков, избегая побочных эффектов и изменяемого состояния.

## Основные принципы

### 1. **Чистые функции**

Функции без побочных эффектов.

```javascript
// ✅ Чистая функция
function add(a, b) {
    return a + b;
}

// ❌ Нечистая функция
let counter = 0;
function increment() {
    counter++; // Побочный эффект
    return counter;
}
```

### 2. **Неизменяемость**

Данные не изменяются, создаются новые.

```javascript
// ❌ Изменение
const arr = [1, 2, 3];
arr.push(4);

// ✅ Неизменяемость
const arr = [1, 2, 3];
const newArr = [...arr, 4];
```

### 3. **Функции высшего порядка**

Функции, которые принимают или возвращают функции.

```javascript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(x => x * 2);
```

### 4. **Рекурсия**

Использование рекурсии вместо циклов.

```javascript
function factorial(n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
```

## Преимущества

### 1. **Предсказуемость**

Чистые функции легче тестировать и понимать.

### 2. **Параллелизм**

Легче распараллеливать выполнение.

### 3. **Отладка**

Меньше скрытых зависимостей.
