# Что такое дескрипторы свойств объектов?

Дескрипторы свойств объектов — это объекты, которые описывают атрибуты свойства объекта в JavaScript. Они позволяют контролировать поведение свойств, такие как возможность записи, перечисления и конфигурирования.

## Что такое дескрипторы свойств?

Дескрипторы свойств — это объекты, которые определяют, как свойство объекта ведет себя при доступе, записи и перечислении.

## Типы дескрипторов

### 1. **Data Descriptor**

Описывает свойство с значением.

```javascript
const obj = {};

Object.defineProperty(obj, 'name', {
    value: 'Иван',
    writable: true,
    enumerable: true,
    configurable: true
});
```

### 2. **Accessor Descriptor**

Описывает свойство с геттером и сеттером.

```javascript
Object.defineProperty(obj, 'age', {
    get() {
        return this._age;
    },
    set(value) {
        if (value > 0) {
            this._age = value;
        }
    },
    enumerable: true,
    configurable: true
});
```

## Атрибуты

### 1. **value**

Значение свойства.

### 2. **writable**

Можно ли изменять значение.

### 3. **enumerable**

Появляется ли в перечислении.

### 4. **configurable**

Можно ли изменять дескриптор.

### 5. **get**

Функция-геттер.

### 6. **set**

Функция-сеттер.

## Заключение

**Дескрипторы свойств — это:**

1. ✅ Объекты, описывающие свойства
2. ✅ Контроль поведения свойств
3. ✅ Data и Accessor дескрипторы

**Атрибуты:**

- value, writable
- enumerable, configurable

**Рекомендации:**

- Используйте для контроля свойств
- Применяйте для валидации
- Используйте для вычисляемых свойств

Дескрипторы свойств — мощный инструмент для контроля объектов.
