# Типы полиморфизма?

Полиморфизм — это способность объектов разных типов обрабатываться через единый интерфейс. Существует несколько типов полиморфизма, каждый со своими особенностями.

## Типы полиморфизма

### 1. **Параметрический полиморфизм (Generics)**

Использование обобщенных типов.

```typescript
function identity<T>(arg: T): T {
    return arg;
}

identity<string>('hello');
identity<number>(42);
```

### 2. **Полиморфизм подтипов (Subtype Polymorphism)**

Объекты подклассов могут использоваться вместо объектов базового класса.

```javascript
class Shape {
    area() {
        throw new Error('Должен быть переопределен');
    }
}

class Circle extends Shape {
    area() {
        return Math.PI * this.radius ** 2;
    }
}

function calculateArea(shape: Shape) {
    return shape.area(); // Полиморфизм
}
```

### 3. **Ad-hoc полиморфизм (Перегрузка)**

Разные реализации для разных типов.

```javascript
// В JavaScript нет перегрузки, но можно эмулировать
function add(a, b) {
    if (typeof a === 'string' && typeof b === 'string') {
        return a + b; // Конкатенация
    }
    return a + b; // Сложение
}
```

### 4. **Полиморфизм времени выполнения (Runtime)**

Выбор метода во время выполнения.

```javascript
class Animal {
    speak() {
        console.log('Звук');
    }
}

class Dog extends Animal {
    speak() {
        console.log('Лай');
    }
}

const animal = new Dog();
animal.speak(); // "Лай" - определяется во время выполнения
```

## Заключение

**Типы полиморфизма:**

1. ✅ Параметрический — Generics
2. ✅ Полиморфизм подтипов — Наследование
3. ✅ Ad-hoc — Перегрузка
4. ✅ Время выполнения — Динамический выбор

**Рекомендации:**

- Используйте полиморфизм подтипов для гибкости
- Применяйте параметрический для обобщений
- Понимайте различия типов

Полиморфизм — ключевой принцип ООП для создания гибкого кода.
