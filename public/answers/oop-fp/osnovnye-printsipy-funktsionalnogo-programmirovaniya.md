# Основные принципы функционального программирования?

Функциональное программирование основано на нескольких ключевых принципах, которые отличают его от других парадигм. Понимание этих принципов важно для эффективного использования ФП.

## Основные принципы

### 1. **Чистые функции**

Функции без побочных эффектов.

```javascript
// ✅ Чистая функция
function add(a, b) {
    return a + b;
}

// ❌ Нечистая функция
let counter = 0;
function increment() {
    counter++; // Побочный эффект
}
```

### 2. **Неизменяемость**

Данные не изменяются, создаются новые.

```javascript
// ❌ Изменение
const arr = [1, 2, 3];
arr.push(4);

// ✅ Неизменяемость
const arr = [1, 2, 3];
const newArr = [...arr, 4];
```

### 3. **Функции высшего порядка**

Функции, работающие с другими функциями.

```javascript
const numbers = [1, 2, 3];
const doubled = numbers.map(x => x * 2);
```

### 4. **Рекурсия**

Использование рекурсии вместо циклов.

```javascript
function factorial(n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
```

### 5. **Композиция функций**

Объединение функций для создания новых.

```javascript
const compose = (f, g) => x => f(g(x));
const add = x => y => x + y;
const multiply = x => y => x * y;
```

## Заключение

**Основные принципы ФП:**

1. ✅ Чистые функции
2. ✅ Неизменяемость
3. ✅ Функции высшего порядка
4. ✅ Рекурсия
5. ✅ Композиция

**Рекомендации:**

- Используйте чистые функции
- Избегайте мутаций
- Применяйте функции высшего порядка
- Используйте рекурсию и композицию

Эти принципы — основа функционального программирования.
