# Что такое статический метод класса (`static`)? Как осуществляется его вызов?

Статический метод — это метод, который принадлежит классу, а не экземпляру класса. Статические методы вызываются напрямую через класс, а не через экземпляр.

## Что такое статический метод?

Статический метод — это метод, который определен на уровне класса и не требует создания экземпляра для вызова.

## Примеры

### Определение статического метода:

```javascript
class MathUtils {
    static add(a, b) {
        return a + b;
    }
    
    static multiply(a, b) {
        return a * b;
    }
}
```

### Вызов статического метода:

```javascript
// Вызывается через класс
MathUtils.add(2, 3); // 5
MathUtils.multiply(2, 3); // 6

// ❌ Нельзя вызвать через экземпляр
const utils = new MathUtils();
utils.add(2, 3); // TypeError
```

## Особенности

### 1. **Нет доступа к `this`**

Статические методы не имеют доступа к `this` экземпляра.

### 2. **Принадлежность классу**

Метод принадлежит классу, а не экземпляру.

### 3. **Утилитарные функции**

Часто используются для утилитарных функций.

## Заключение

**Статический метод — это:**

1. ✅ Метод класса, а не экземпляра
2. ✅ Вызывается через класс
3. ✅ Нет доступа к `this`

**Вызов:**

- `ClassName.methodName()`
- Не через экземпляр

**Рекомендации:**

- Используйте для утилитарных функций
- Применяйте для методов, не требующих экземпляра
- Не используйте `this` в статических методах

Статические методы — полезный инструмент для организации кода.
