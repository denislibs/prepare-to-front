# Какие принципы можно использовать вместе с наследованием?

Наследование можно комбинировать с другими принципами ООП и проектирования для создания более гибкого и поддерживаемого кода. Понимание этих комбинаций важно для эффективного проектирования.

## Принципы, используемые с наследованием

### 1. **Композиция**

Использование композиции вместе с наследованием.

```javascript
class Engine {
    start() {
        console.log('Двигатель запущен');
    }
}

class Car {
    constructor() {
        this.engine = new Engine(); // Композиция
    }
    
    start() {
        this.engine.start();
    }
}
```

### 2. **Полиморфизм**

Использование полиморфизма с наследованием.

```javascript
class Shape {
    area() {
        throw new Error('Должен быть переопределен');
    }
}

class Circle extends Shape {
    area() {
        return Math.PI * this.radius ** 2;
    }
}
```

### 3. **Инкапсуляция**

Защита данных в иерархии наследования.

```javascript
class Animal {
    #name; // Приватное поле
    
    constructor(name) {
        this.#name = name;
    }
    
    getName() {
        return this.#name;
    }
}
```

### 4. **Абстракция**

Использование абстрактных классов.

```javascript
class Vehicle {
    start() {
        throw new Error('Должен быть реализован');
    }
}
```

## Заключение

**Принципы с наследованием:**

1. ✅ Композиция
2. ✅ Полиморфизм
3. ✅ Инкапсуляция
4. ✅ Абстракция

**Рекомендации:**

- Комбинируйте наследование с композицией
- Используйте полиморфизм
- Применяйте инкапсуляцию
- Используйте абстракцию

Комбинация принципов создает более гибкий код.
