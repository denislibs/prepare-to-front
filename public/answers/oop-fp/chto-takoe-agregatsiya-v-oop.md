# Что такое агрегация в ООП?

Агрегация — это тип отношения "имеет" в ООП, при котором один объект содержит ссылку на другой объект, но оба объекта могут существовать независимо. Это слабая форма композиции.

## Что такое агрегация?

Агрегация — это отношение, при котором объект содержит другой объект, но оба могут существовать независимо друг от друга.

## Пример

```javascript
class Department {
    constructor(name) {
        this.name = name;
        this.employees = []; // Агрегация
    }
    
    addEmployee(employee) {
        this.employees.push(employee);
    }
}

class Employee {
    constructor(name) {
        this.name = name;
    }
}

const employee = new Employee('Иван');
const department = new Department('IT');
department.addEmployee(employee);

// Employee может существовать без Department
// Department может существовать без Employee
```

## Характеристики

### 1. **Слабая связь**

Объекты слабо связаны.

### 2. **Независимое существование**

Объекты могут существовать независимо.

### 3. **Отношение "имеет"**

Один объект имеет другой.

## Отличие от композиции

| Характеристика | Агрегация | Композиция |
|----------------|-----------|------------|
| Связь | Слабая | Сильная |
| Существование | Независимое | Зависимое |

## Заключение

**Агрегация — это:**

1. ✅ Отношение "имеет"
2. ✅ Слабая связь
3. ✅ Независимое существование

**Характеристики:**

- Слабая связь
- Независимое существование
- Отношение "имеет"

**Рекомендации:**

- Используйте для слабых связей
- Применяйте когда объекты независимы
- Различайте от композиции
