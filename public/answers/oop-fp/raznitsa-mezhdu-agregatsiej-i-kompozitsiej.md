# Разница между агрегацией и композицией?

Агрегация и композиция — это два типа отношений "имеет" в ООП. Понимание различий важно для правильного моделирования отношений между объектами.

## Агрегация

### Характеристики:

- Слабая связь
- Объект может существовать независимо
- Отношение "имеет"
- Жизненный цикл независим

```javascript
class Department {
    constructor(name) {
        this.name = name;
        this.employees = []; // Агрегация
    }
    
    addEmployee(employee) {
        this.employees.push(employee);
    }
}

class Employee {
    constructor(name) {
        this.name = name;
    }
}

const employee = new Employee('Иван');
const department = new Department('IT');
department.addEmployee(employee);
// Employee может существовать без Department
```

## Композиция

### Характеристики:

- Сильная связь
- Объект не может существовать без родителя
- Отношение "содержит"
- Жизненный цикл зависим

```javascript
class Car {
    constructor() {
        this.engine = new Engine(); // Композиция
    }
}

class Engine {
    constructor() {
        // Engine создается внутри Car
    }
}

const car = new Car();
// Engine не может существовать без Car
```

## Различия

| Характеристика | Агрегация | Композиция |
|----------------|-----------|------------|
| Связь | Слабая | Сильная |
| Существование | Независимое | Зависимое |
| Жизненный цикл | Независимый | Зависимый |

## Заключение

**Агрегация:**

1. ✅ Слабая связь
2. ✅ Независимое существование
3. ✅ Отношение "имеет"

**Композиция:**

1. ✅ Сильная связь
2. ✅ Зависимое существование
3. ✅ Отношение "содержит"

**Рекомендации:**

- Используйте агрегацию для слабых связей
- Используйте композицию для сильных связей
- Понимайте жизненный цикл объектов

Правильное различение помогает в моделировании отношений.
